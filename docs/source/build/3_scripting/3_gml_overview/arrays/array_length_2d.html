array_length_2d


描述

使用该函数你可以得到一个二维数组的高度（以数组元素的个数形式）。你需要提供数组第一个维度的元素索引，然后函数会返回你在该元素下数组的长度（如果要获得数组的第一个维度请使用函数array_height_2d）。The function will return 0 if the variable given is not an array or 1 if the variable is a 1D array (as there is still 1 row).


语法：

array_length_2d(array_index, n);


参数：

参数 描述
array_index 待检测的数组索引。
n 待获取数组长度的数组元素。

返回：

实数


举例：

for (var i = array_height_2d(a) - 1; i > -1; i--;)
   {
   for (var j = array_length_2d(a, i) - 1; j > -1; j--;)
      {
      a[i, j] = -1;
      }
   }

上述代码将会遍历该二维数组并将每一元素设为-1。
