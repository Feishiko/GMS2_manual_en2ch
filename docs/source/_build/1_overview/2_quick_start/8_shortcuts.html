快捷键


在 GameMaker Studio 2 有很多可供使用的键盘快捷键，并且尽管它们当中的绝大多数在IDE中相关菜单项的旁边被标注出来了，但对于用户来说并不是所有快捷键在开始的时候很显眼。在这里你可以查阅键盘快捷键的完整列表作为一个简单参考向导。一般地，快捷键被分为如下三类：

  • 全局 - 全局快捷键是那些你在 GameMaker Studio 2 IDE 中无论什么位置都能生效的快捷键，除非你选择了一个覆盖了它的某个窗口或部件.

  • 窗口 - 窗口快捷键是那些专门用于单独窗口或标签页的快捷键，除非部件的快捷键覆盖了该快捷键。

  • 部件 - 部件快捷键是仅针对于单独一个控件部件（例如，取色器）生效的快捷键
