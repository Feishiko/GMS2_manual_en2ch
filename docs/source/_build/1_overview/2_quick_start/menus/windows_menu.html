窗口菜单（Windows）


该菜单用于打开常用的 IDE 停靠窗口。由于停靠栏可以被重新安置，且其内容可以被作为为一个独立窗口而移除，甚至可以彻底关闭，因此该窗口允许你在你发现你还需要它们的时候重新打开它们。可以从这里打开的窗口有 资源 窗口和不同的 输出 窗口（输出，搜索，SCM，断点和编译/语法错误）。

接下来，你可以打开 最近窗口 列表，还可以打开 环境变量 窗口，在其中可以看到“首选项”中指向的各种环境变量的不同路径。这些变量不能被编辑（名字和路径/设定都不可以），但是你可以选取复制它们并可以将其粘贴到别的地方以供使用。

最后你还有一个打开 教程窗口 的选项。该选项将会重新打开任何与你所打开的教程项目有关的窗口，但是，前提是该项目必须已经创建为教程的一部分（否则该选项不可用）。关于创建教程项目的信息，请参见 市场 章节（因为所有教程的创建、管理以及下载都来自于这里）。
