工具菜单（Tools）


该菜单允许你在GameMaker Studio 2中打开各种可用的独立工具，具有如下选项：

  • 纹理组（Texture Groups） - 查看纹理组窗口。

  • 音频组（Audio Groups） - 查看音频组窗口

  • 混音器（Sound Mixer） - 查看混音器窗口。

  • 生成项目图（Generate Project Images） - 打开生成项目图窗口。

  • To-do List - Opens the To-do List editor window.
