市场菜单（Marketplace）


除了为你的项目创建你自己的资源和代码以外，你也可以通过 GameMaker Studio 2 的 市场（Marketplace）来下载免费的和付费的资源并将其添加到你的项目中。在 GameMaker 市场 一节中你可以找到更多关于市场的信息，此菜单的选项如下所示：

  • 前往市场（Go to the Marketplace） - 打开市场窗口，在这个窗口里你可以浏览资源。

  • 我的库（My Library） - 查看你当前的市场内容库。

  • 我的购买历史记录（My Purchase History） - 该选项将会打开一个浏览器并带你到你的账户页面，在这个页面里你可以看到你为此账户购买的历史记录。

  • 更新现有的资源包 - 该选项将会打开 资源包更新向导（Update Package Wizard）以便你能够更新一个现有的市场资源包。

  • 创建资源报（Create Package） - 使用该选项你可以打开 资源包创建向导（Create Package Wizard）来为市场创建一个新的资源包或教程。
