工作流程


在快速入门指南的这个部分，我们将带领你完成创建精灵、对象和房间，直到你可以编译项目并查看结果。正如你所见，GameMaker Studio 2 在许多方面都具有创新性，因此我们希望这部分能够向你提供足够的知识，以深入了解GMS2并开始工作，而不会遇到太多问题...

为了说明整个工作流程，我们将做一个非常简单的项目：让一个球在房间里弹跳。在这个过程中，你需要使用图像编辑器、物体编辑器、代码编辑器和房间编辑器，事实上你创建任何项目都将花费大部分时间在这几个编辑器上。由于这是引导你入门的迷你教程，我们已经将它分割成如下几个小节：

  1. 创建精灵
  2. 创建图块集
  3. 创建声音
  4. 创建物体
  5. 创建房间
