创建声音


所有游戏都必须有音效，大多数游戏还有音乐，这意味着你需要在 GameMaker Studio 2 中为游戏添加音效以及音乐。这些操作是通过 声音编辑器 来完成的，需要你创建一个新的声音资源。

GameMaker Studio 2 中的声音可以是压缩或未压缩的，压缩的声音在运行库中有更多的选项来进行处理。未压缩的声音是保存为 WAV 格式文件的声音，而压缩后的声音一般是 MP3 或 OGG 格式。一般来说，小规格的声音效果一般是 WAV 格式，而大规格的声音效果以及音乐则使用 MP3 或 OGG 格式。与其他的资源一样，建议您使用合适的前缀（如“snd_”）来命名您的声音，以便以后可以轻松识别。

添加声音非常简单，只需单击 添加声音按钮，然后在打开的文件浏览器中浏览您要使用的文件。一旦你选择了文件并加载，你就可以使用显示中的播放控件预览它，并改变文件的某些属性例如应该如何导出等。如果您不确定这一点，那么只需选择为“按原样”即可，但如果您要添加音乐或 MP3 格式的声音，则应该更改为 压缩 - 非流式 格式。你可以在 这里 找到更多关于声音编辑器的信息。

请注意，声音在磁盘中会占用大量空间，所以在创建游戏项目时应该注意。你真的需要一个只有几分之一秒的 48KHz 音效吗？可能并不是这样，考虑将采样率降至 11025 或 22050 以获得相近的效果，对于音乐或循环效果等持续时间较长的音效，也可考虑使用压缩格式。还要注意的是，如果您打算使用3D音效，则初始声音文件 必须 是单声道的。

你现在可以关闭声音编辑器了，我们来继续创建物体...
