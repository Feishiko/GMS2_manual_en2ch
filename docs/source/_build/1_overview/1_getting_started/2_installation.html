安装


GameMaker Studio 2 被设计用于具有如下最低配置的台式计算机上：

对于 Windows 平台你至少需要以下配置：

  • 64bit Intel 可兼容双核 CPU
  • 2GB RAM
  • 兼容DX11的显卡
  • Microsoft Windows 7 64位
  • 至少3GB硬盘空间

而我们推荐的配置是：

  • 64bit Intel 可兼容四核 CPU
  • 8GB RAM
  • 基于DX11的显卡
  • Microsoft Windows 10 64位
  • SSD硬盘

对于Mac OS平台，你至少需要具有以下最低规格：

  • Mac OSX El Capitan 或更高
  • 一个 Intel 双核 i3 处理器
  • 4GB或更多RAM
  • Xcode 7或更高

而我们推荐的配置是：

  • macOS
  • 一个 Intel 四核 i5 处理器
  • 8GB RAM
  • SSD硬盘

你可以从 YoYo Games 网站下载GameMaker Studio 2的安装包。一经安装后首次运行GameMaker Studio 2，你可能会在使用产品前被提示升级。在所有更新都安装后，在使用产品之前你将会被要求为你的YoYoGames 账户（参见这这里）提供许可的详细信息，这意味着初次使用GameMaker Studio 2的时候，你必须联网经过初始登录后，你应该就可以离线地使用本产品了，但是它会定期地要求你联网来验证登录账户。
