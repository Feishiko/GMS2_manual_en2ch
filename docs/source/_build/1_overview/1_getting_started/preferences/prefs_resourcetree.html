偏好设置 - 资源树


资源树偏好设置 用于设置资源树及其元素功能的某些属性。主要选项有：

  • 在没有警告的情况下同时打开的最大资源数：你可以使用 / + 一次选择多个资源，然后单击 并选择全部打开。如果你选中的数目超过选项中的数目，你将会看到警告信息，因为同时打开太多的资源在低配电脑上会出出一些性能上的问题。默认值为20。

  • 双击精灵打开图片编辑器：默认情况下，双击 任何资源都将打开该资源的编辑器，但对于精灵，在创建新资源并初始设置后，双击打开它进入编辑图像界面而不是编辑精灵属性。选中这个选项将打开这个精灵资源的图像编辑器，而不是精灵资源本身。你仍然可以通过右键点击 并选择 打开属性 打开精灵编辑器。该选项默认为关。

  • 双击物体打开所有的物体事件：默认情况下，双击 任何资源都将打开该资源的编辑器，但对于物体，经常有这种情况，在创建新资源并初始设置后，双击打开它只是编辑事件而不是物体属性。选择此选项，双击将在新的工作区打开物体的不同事件，在新的工作区每个事件都会有自己的选项卡。你仍然可以通过右键单击 并选择 打开属性 来打开物体编辑器。该选项默认为关。

  • 字体资源可以缩小到 100% 以下：若你打开此选项，然后更改资源项目大小，当设置为小于100％时你可以强制 GameMaker Studio 2 缩小资源树中的文本以及图标。 默认情况下此选项关闭，资源树文本只 能扩大 不能缩小。

  • 字体资源可以放大到 100% 以上：若你打开此选项，然后更改资源项目大小，当设置为大于100％时你可以强制 GameMaker Studio 2 放大文本以及资源树中项目的图标。默认情况下打开，资源树文本可以 扩大。

  • 默认缩放（%）：设置启动新项目时资源树的默认缩放百分比。默认值为100％。

  • 最小缩放倍数（%）：设置资源树允许的最小缩放值。默认值为50％。

  • 最大缩放倍数（%）：设置资源树允许的最大缩放值。默认值为250％。

  • 显示水平分割线：你可以在此处启用/禁用资源树中不同资源类型之间的水平分隔符。默认情况下开启以显示分隔符。


  • 显示树状线：你可以在此处 启用/禁用 在资源树中的资源一侧显示的树状线。 默认情况下显示树状线。


  • 显示组节点图标：启用或禁用资源名称旁边显示的每个资源类型的图标。该选项默认关闭。

在显示资源树时，可以更改 GameMaker Studio 2 使用的配色方案。本节分为以下三部分：

  • 背景： 此部分允许你设置资源树中主要资源节点的背景颜色。 通过点击任何一个选项，你都可以打开颜色选择器，你可以在其中选择要与常规 IDE 节点颜色混合的颜色和 alpha值。 每个的默认值为$ FFFFFFFF。
  • 图标： 此部分允许你设置用于表示不同资源树资源类型的文件夹图标的颜色和 alpha值。 每个的默认值为$ FFFFFFFF。请注意，如果将Alpha值设置为0，则不会看到这些图标。

  • 标签： “标签”是用于描述或命名资源树中的节点或项目的文本，此处可以更改文本的颜色以及字体和样式。 你应该首先更改 默认 设置以创建所有资源树节点，然后继续根据需要编辑单个节点标签设置（请注意，它们的初始选择都是“使用默认值”）。

  • 杂项： 这些选项影响资源树的某些全局颜色属性：
    • 水平分割线颜色：在树中分割每个离散资源类型的线条的颜色。默认颜色为 $3E3E3EFF。

    • 树状线颜色：将用于绘制不同资源节点之间的树状线的颜色。默认颜色为 $666666FF。

    • 节点选择高亮：用于突出资源树中选定节点的颜色。默认颜色为 $039D5BFF。

在处理资源树的资源的情况，这里可以选择确认对话框的行为。可用的选项有：

  • 自动响应资源删除：从资源树中删除资源时，通常会收到一条警告，询问你是否确定要继续。通过将此选项设置为“是”，消息将被取消显示，并像你点击“确定”按钮一样继续。默认值为“显示消息”。

  • 自动响应打开多个资源：就像上面提到的，当从资源树中一次打开多个资源时，如果打开数目超过指定的数字，则会收到警告。通过将此选项设置为“是”，警告将被取消显示，并像你点击“确定”按钮一样继续。默认值为“显示消息”。

  • 通过节点拖动自动响应设置房间父级：当你拖动房间到资源树中的另一个房间下面，然后释放它，系统将显示一条消息，询问你是否希望让被拖动的房间成为拖动位置所在房间的子房间。通过将此选项设置为“是”，警告将被取消显示，并像你点击“确定”按钮一样继续。默认值为“显示消息”。

  • 自动响应删除自定义视图：创建自定义资源树视图后，当你删除它时，系统将显示一条消息，询问是否继续并删除该视图。通过将此选项设置为“是”，警告将被取消显示，并像你点击“确定”按钮一样继续。默认值为“显示消息”。

  • 自动响应 DnD™ 转化为 GameMaker Language：选择将 DnD™ 节点转换为 GML 时，系统将提示你是否确认更改，因为该动作无法撤消。通过将此选项设置为“是”，警告将被取消显示，并像你点击“确定”按钮一样继续。默认值为“显示消息”。
