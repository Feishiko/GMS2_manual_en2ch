偏好设置 - 物体编辑器


物体编辑器偏好设置 用于定义 物体编辑器 窗口的特定属性。主要部分包括以下选项：

  • 添加默认的事件内容：该选项默认启用，它所做的只是为所有新事件在 代码编辑器 中添加一个简单的注释，以提醒你可以在那里添加代码。

  • 默认的事件内容：在这里你可以编辑被添加到新事件的代码编辑器窗口的默认注释。

物体编辑器还存在下面的子类别：

这一部分是使用物体编辑器时可能会显示的对话框，有下面可用的选项：

  • 自动响应 DnD™ 转换为 GML：你可以在拖拽工作区中使用鼠标右键 选择将事件的所有 DnD 转换为 GML 代码。此处的默认操作是显示一条警告消息，询问你是否要继续，但你可以将其设置为“确定（Okay）”，并且转换前不会显示该对话框。

  • 自动响应超出物理顶点限制：当物理形状编辑器中超过允许的顶点数目时会显示一条警告信息。这里默认的动作是显示警告信息，但是你可以设置“确定（Okay）”让对话框不显示。
