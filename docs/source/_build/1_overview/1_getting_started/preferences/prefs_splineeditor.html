偏好设置 - 样条曲线（Spline）编辑器


样条曲线编辑器偏好设置主要用来定义路径编辑器的外观和感受，尽管也将会影响IDE的特定的其它方面（比如物理形状编辑器）。存在以下选项：

  • 插入路径点精度（像素） - 设置在路径上插入点的精度值（以像素为单位），即：鼠标必须在此精度范围内才能插入点，而不是添加新的点。默认20px。

  • 锚点宽度 - 设置锚点的宽度，以像素为单位。默认8px。

  • 锚点高度 - 设置锚点的高度，以像素为单位。默认8px。

  • 路径颜色 - 设置在编辑路径时用于显示路径的颜色。默认值是（rgba）#FFFFFFFF。

  • 高亮路径段颜色 - 用于设置高亮显示所选路径段的颜色。默认值是（rgba）#FFFFFFFF。

  • 无效路径颜色 - 用于设置突出显示无效路径的颜色。默认值为（RGBA颜色） #FF0000FF。

  • 锚点颜色 - 设置用于绘制路径锚点的颜色。默认值是（rgba）#C0C0C0FF。

  • 无效锚点颜色 - 设置用于绘制路径无效锚点的颜色。默认值是（rgba）#800000FF。

  • 选中的锚点颜色 - 设置用于绘制已突出显示（选择）的路径锚点的颜色。默认值是（rgba）#00A4FFFF。

  • 选中的无效锚点颜色 - 设置用于绘制突出显示但却无效的锚点的颜色。默认值是（rgba）#FFA400FF。

  • 鼠标悬停的锚点颜色 - 设置用于绘制活动锚点的颜色。默认值是（rgba）#FFFFFFFF。

  • 鼠标悬停的无效锚点颜色 - 设置用于绘制活动但无效的锚点的颜色。默认值为（RGBA颜色） #FF0000FF。

  • 蒙版覆盖区域颜色 - 如果启用蒙版，请设置蒙版的覆盖颜色（见上文）。默认值为（RGBA颜色） #404040FF。
