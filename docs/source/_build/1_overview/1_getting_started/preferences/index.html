常规偏好设置

偏好设置窗口 是 GameMaker Studio 2的一个重要的组成部分，因为在这里你能够设置并自定义你的IDE使用体验还可以设置任意平台的SDK等……在这一部分，我们将会讨论偏好设置窗口中IDE的具体部分——对于单独平台请参阅章节目标平台（Target Platform）偏好设置请注意，从此窗口修改任何偏好设置你都需要点击应用（Apply）按钮来生效。

如果，在依据你的个人喜好改动过这里的选项来自定义GameMaker Studio 2之后，你希望备份你所设定的偏好设置，那么你需要前往文件夹%AppData%/GameMakerStudio2/<your_account_name>/，并将下面两个文件复制到比较安全的位置：

  • local_settings.json
  • cloud_settings.json

而如果你希望重置你所修改的偏好设置，那么你需要点击窗口顶端的恢复为默认状态（Restore Defaults）请注意，这只会将当前正在编辑的部分恢复为默认。

你可以通过IDE顶端的文件（File）菜单来打开GameMaker Studio 2的偏好设置：

一旦偏好设置窗口打开，你会有如下的各种选项：

  1. 常规设置
  2. 拖拽按钮（DnD）
  3. 图像编辑器
  4. 精灵编辑器
  5. 物体编辑器
  6. 房间编辑器
  7. 样条曲线编辑器
  8. 图块集编辑器（Tile Set Editor）
  9. 时间轴
  10. 扩展编辑器
  11. 资源树
  12. 文本编辑器
  13. 语言
  14. 市场（Marketplace）
  15. 插件
  16. 运行库管理

请注意，还有一个部分叫做特定平台偏好设置（Platform Specific Preferences）被列举在手册的这个位置。
