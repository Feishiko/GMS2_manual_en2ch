授权


一旦你下载并安装了GameMaker Studio 2，首次使用你会被要求输入你的许可证详细信息：

你的许可证是和YoYo 账户 (YYA) 绑定的，并且你必须有一个许可证才能使用GameMaker Studio 2。此帐户包含你拥有的YoYo Games产品的所有许可详细信息，它也用于YoYo Games 市场。你可以在下面的页面中找到关于YYA的详细信息。

  1. 创建一个YoYo账户
重要！ 如果你已经在使用一个 GameMaker Studio 2 的试用版许可证，然后买一个其它的许可证（桌面版、移动版、UWP、网页版或者控制台），然后你可能 需要登出IDE，接着重启GameMaker Studio 2，之后再次登录以确保新的许可证正确应用（尽管这应该会自动生效）。你可以从 文件菜单 中登出。


代理支持

如果你需要为 GameMaker Studio 2 设置代理，可以点击 代理 来添加。这将弹出如下图窗口：

现在你可以添加你的代理的详细信息。上图所示的一些字段是可选的，如果不需要某个字段，那么只需要将字段留空即可。在继续之前，如果你已经在 Windows Internet 选项设置了代理，并且它不需要用户名和密码，那么你不需要在 GameMaker Studio 2 设置代理。

如果你的代理需要用户名和密码，那么请将它们填入上图字段中。注意，在这种情况下代理、用户名、密码 必须 填写，代理 IP 应该添加端口号，例如： xx.xx.xx.xx:port.

一旦你添加了相关信息，就可以点击 确定 并且登陆 GameMaker Studio 2。
