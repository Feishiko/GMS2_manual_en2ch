微型网络服务器


微型网络服务器（Micro Web Server,MWS）是一个主要用于连接 GameMaker Studio 2 可用的目标平台（如 HTML5 平台）及向目标平台添加伺服文件的小工具。当你在需要使用微型网络服务器的目标上运行游戏的时候，你会看到当游戏在所选设备上被伺服时会打开一个新的窗口：

注意：当你首次通过 MWS 在目标平台上运行游戏时可能会得到防火墙警报。如果发生这种情况，你应该允许网络服务器穿过防火墙进行通信。


这个网络服务器会显示一定量的与所伺服文件相关的调试信息，以及底部的一个用来清空这些信息的按钮。在“清除”按钮旁边是项目 JS 文件根目录的路径，而在它下面有 网络速度（Network Speed） 选项。默认设置为“没有限制”，但点击它将打开一个不同的网络选项列表：

更改此设置还会更改右侧的包 大小 选项。你还可以手动编辑它，在这种情况下，网络速度将被设置为“Custom”。将其设置为除最大值之外的任何值，将有效地控制网络传输速度，并允许你在低带宽环境中测试游戏。

你还可以看到允许的 IP 列表以及正在使用的端口号，但是这些不能在微型网络服务器中进行编辑，应该在给定设备目标的 目标管理器设备 设置中进行设置。
