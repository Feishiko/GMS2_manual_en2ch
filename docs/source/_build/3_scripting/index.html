脚本

这一部分讨论在GameMaker Studio 2中可用于编程的丰富的脚本语言。创建你的项目使用的语言或者方法取决于你的技能和你以前的技术背景，既然每个人都是不同的， GameMaker Studio 2 致力于满足你的不同需求，为你打造游戏提供了两种不同的选项：

拖拽式操作（DnD™）适合初学者或者那些专业偏向视觉或艺术的开发者（这有时也可以称为“视觉脚本语言”）。由拖拽 动作 组成，拖放到物体编辑器中，创建许多物体实例将遵循的一系列函数。在手册的以下章节，你可以找到 DnD™ 的使用方法：
  1. DnD 概述
  2. DnD 参考

在这里你可以看到 GameMaker 语言语法的概述，以及使用方法和程序结构的例子，你还可以找到所有内置 GML 函数的完整的参考指南。如果你是脚本使用的新手，建议从 GML 概述开始，如果不是，只需要使用参考指南去查找所需要的函数名就可以了：
  1. GML 概述
  2. GML 参考

值得注意的是，在创建游戏时，你实际上可以 “混合搭配” 这些不同的编程技术，并在创建新项目时选择一个或另一个并不限制你 只 使用那种语言。
