game_end


语法：

game_end();


返回： N/A（无返回值）


描述

使用此函数，你可以结束游戏（并且将触发 游戏结束事件）。这不会立即发生，而是在当前步结束时发生，因此调用此函数后在同一步中的任何代码仍将运行。请注意，此函数具有以下限制：

  • 在 Android（和 Amazon Fire）设备上，调用 game_end() 会将应用程序推送到后台，但 不 关闭应用程序。这必须由用户完成。

  • 在 iOS 上，它什么都不做，然后默默地失败。

  • 在 UWP，XBoxOne，PS4 和 Nintendo Switch 上，它会使游戏崩溃，因此 不得使用。

  • 在 HTML5 上，它将结束游戏，但让用户盯着网页上的空白绘制画布，因此不建议在目标平台上使用它。

  • 在 Windows，Linux 和 Mac 上，该函数只是结束游戏并关闭游戏窗口（将触发 游戏结束事件）。


例如：

if keyboard_check_pressed(vk_escape) game_end();

如果玩家按下 “Escape（退格）” 键，这将结束游戏。
