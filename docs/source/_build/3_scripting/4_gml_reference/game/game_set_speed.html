game_set_speed


描述

此函数可用于设置游戏速度。以两种方式之一设置 - 每秒游戏帧数（FPS）或每游戏帧微秒数（MPF）- 你可以使用以下两种方式之一：

常量 描述
gamespeed_fps 使用帧数 / 秒设置游戏速度。
gamespeed_microseconds 使用微秒 / 帧设置游戏速度。


因此，例如，30 帧 / 秒的游戏速度与 33333 微秒 / 帧的游戏速度相同，用函数表示为：

game_set_speed(30, gamespeed_fps);

或：

game_set_speed(33333, gamespeed_microseconds);


语法：

game_set_speed(speed, type);

参数 描述
speed 新的游戏速度（FPS 或 MPF）。
type 用于设置游戏速度的方法类型（参见上面的常量）。


返回：

N/A（无返回值）


例如：

if os_browser == browser_not_a_browser
    {
    game_set_speed(60, gamespeed_fps);
    }
else
    {
    game_set_speed(30, gamespeed_fps);
    }

上面的代码检查游戏是否在浏览器中运行，并将游戏速度相应地设置为 FPS 值。
