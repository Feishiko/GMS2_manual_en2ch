game_load_buffer

语法：

game_load_buffer(buffer);


参数 描述
buffer 要加载的缓冲区 ID。


返回： N/A（无返回值）


描述

使用此函数，你可以加载之前已保存的游戏状态。游戏是从先前创建的 “增长” 缓冲区加载的（参见 缓冲区），缓冲区必须使用 game_save_buffer保存游戏状态函数。


例如：

if keyboard_check_pressed(ord("L"))
   {
   if global.Checkpoint game_load_buffer(save_buff);
   }

当播放器按下 “L” 键时，上述代码将从变量 “save_buff” 中索引的缓冲区加载先前保存的游戏状态，仅当全局变量为 true 时。
