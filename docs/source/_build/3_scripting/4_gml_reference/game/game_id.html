game_id

语法：

game_id


返回： Real（实数）


描述

此 只读 变量返回你创建的游戏的唯一标识符。如果你需要唯一的文件名，或者只需要识别你的游戏的任何其他内容，你可以使用此选项。这可以在 游戏选项 中设置。


例如：

ini_open("Score.ini");
ini_write_real("Scores","0",score + game_id);
ini_close();

上面的代码通过在保存之前将 game_id 与分数相加来对分数执行最基本的加密。在回到游戏中读取它时，你将用读取到的值减去 game_id 以再次获得正确的值。
