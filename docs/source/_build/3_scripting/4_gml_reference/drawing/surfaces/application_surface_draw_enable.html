application_surface_draw_enable


语法：

application_surface_draw_enable(flag);

参数 描述
标识 标识可以为 真 (允许，默认值) 或 假 (不允许)。


Returns:

N/A（无返回值）


描述

你可以使用这个函数决定允许或者不允许应用表面层的自动绘制。默认是允许，但是也有许多需要接管何时及怎样进行表面绘制的情况 (比如使用着色器的时候)，这里你可以用这个函数设置为假，自主决定绘制的时间和方式。


例如：

application_surface_draw_enable(false);

上述代码将切换应用表面层的自动绘制为假。


返回: 表面层
