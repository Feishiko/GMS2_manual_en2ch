colour_get_red

返回所给定颜色的红色分量。

语法：

colour_get_red(col);


参数 描述
col 待检查的颜色


返回： Real（实数）


描述

这个函数返回给定颜色的红色值，这个值在0到255之间，0代表不含有红色，255代表100%的红色。下面的图像说明这一点:


举例：

r_comp = colour_get_red(c_teal);

上述代码获取颜色常量 "c_teal"的红色分量，并且将其储存到变量 "r_comp"中。
