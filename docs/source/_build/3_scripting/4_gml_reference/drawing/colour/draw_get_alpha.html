draw_get_alpha

获取当前的绘制透明度值。

Syntax:

draw_get_alpha()


返回： Real（实数）


描述

这个函数返回当前的绘制透明度值，这个数值的范围是从0（完全透明）到1（完全不透明）。这个绘制透明度会影响所有的绘制函数，你可以使用 draw_set_alpha 函数来修改它。


例如：

if draw_get_alpha() != 1
   {
   draw_set_alpha(1);
   }

上述代码首先检测当前绘制透明度，当其不为1时，将其设为1。
