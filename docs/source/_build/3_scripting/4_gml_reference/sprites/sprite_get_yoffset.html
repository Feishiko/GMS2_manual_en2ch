sprite_get_yoffset


Syntax:

sprite_get_yoffset(index);

参数 描述
index 将被获取原点Y坐标的精灵的索引。


返回

实数


描述

当你在精灵编辑器中设置精灵时，你需要为精灵设置原点 。原点就是此精灵被接触或绘制的点。此函数将会返回原点的Y轴坐标，此坐标是相对于精灵的左上角（0，0）点的。下面的图像说明这一点:


例如：

if y - sprite_get_yoffset(sprite_index) < 0
   {
   y = sprite_get_yoffset(sprite_index);
   }

上述代码将会确保实例将会存在房间中，这样精灵就始终可见。
