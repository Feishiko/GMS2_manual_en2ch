sprite_flush_multi


描述

使用此函数你可以移除任意数量的精灵纹理，和它们所在的纹理页——从显存中移除。纹理页将在下次使用时被加载。你需要提供精灵资源的索引数组，然后它们将在调用此函数时被从存储中移除。 默认情况下 ，此函数在windows平台（YYC和标准）没什么用，因为所有纹理将在一开始就被加载进存储中。. 如果你在制作跨平台游戏，你在windows平台中不管此函数，它将不会起任何作用，也不报任何错。但你可以强迫windows平台在一开始不加载任何文件，那么你就可像在其它平台上那样在windows平台上使用清除和预加载函数。此设置可以在 Windows Options 窗口中找到。


语法：

sprite_flush_multi(array);


参数：

参数 描述
array Array with the sprite indices to flush


返回：

N/A（无返回值）


举例：

spr_a[0] = spr_Player_Aura1;
spr_a[1] = spr_Player_Aura2;
spr_a[2] = spr_Player_Aura3;
spr_a[3] = spr_Player_Aura4;
sprite_flush_multi(spr_a);

上述函数将会创建一个数组，数组中每个元素都存储一个精灵索引。此数组将会被用来从存储中清除那些精灵纹理。
