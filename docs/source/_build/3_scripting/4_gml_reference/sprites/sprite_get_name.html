sprite_get_name


语法：

sprite_get_name(index);

参数 描述
index 将会被获取名称的精灵的索引。


返回

字符串


描述

此函数将把特定精灵的名称 作为字符串 返回。这个名称是指在GameMaker Studio 2 主窗口的资源树中的特定名称 。请注意，此函数只返回字符串，你不能使用名称调用精灵，而需要使用精灵索引来调用。你可以在 asset_get_index()函数中使用名称字符串来获取精灵索引。


例如：

str = sprite_get_name(sprite_index);

上述代码将会根据精灵索引获得精灵名称并将其存储在变量 "str"中。
