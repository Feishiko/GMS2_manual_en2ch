sprite_set_cache_size_ext


Syntax:

sprite_set_cache_size_ext(ind, index, max);

参数 描述
ind 将要被改变存储大小的精灵的索引。
index 将要被改变存储大小的次级图像的精灵。
max 此精灵可存储的最大复制的数量。


返回

N/A（无返回值）


描述

当你执行图像混合操作后，HTML5无法合适地执行它。. 因此 GameMaker Studio 2 将会暂时存储一个图像的混合复制品并加载它。此函数将会设置在原有精灵被重写前，可以存储多少给定精灵的混合复制品。默认是4。此函数只对精灵的一个给定次级图像有效。


例如：

sprite_set_cache_size_ext(sprite0, 0, 2);

上述代码将会把sprite0的第一个次级的图像的存储限制设为2。
