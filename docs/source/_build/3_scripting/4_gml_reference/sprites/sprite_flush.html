sprite_flush


描述

使用此函数你可以移除给定的精灵纹理，和它们所在的纹理页——从显存中移除。纹理页将在下次使用时被加载。你要提供在创建精灵资源时定义的精灵索引，然后此精灵所在的纹理页将会被从纹理存储中移除。. 默认情况下 ，此函数在windows平台（YYC和标准）没什么用，因为所有纹理将在一开始就被加载进存储中。. 如果你在制作跨平台游戏，你在windows平台中不管此函数，它将不会起任何作用，也不报任何错。但你可以强迫windows平台在一开始不加载任何文件，那么你就可像在其它平台上那样在windows平台上使用清除和预加载函数。此设置可以在 Windows Options 窗口中找到。

注意： 在多数设备上，将纹理放进纹理存储会暂时降低性能。所以推荐你使用 <t1>sprite_preftech</t1> 在游戏关卡开始就预读取需要用到的图像来避免性能降低。


语法：

sprite_flush(ind)


参数：

参数 描述
ind The index (resource name) of the sprite asset to flush


返回：

N/A（无返回值）


举例：

sprite_flush(spr_Player_Aura);

上述函数将会从存储中清除 "spr_Player_Aura" 。
