matrix_get


描述

此函数返回 16 个值的一维数组，对应于给定的 4 x 4 矩阵类型，其中第 1 行是元素 [0 - 3]，第 2 行是元素 [4 - 7] 等...（参见 主页 图像）。可用的矩阵是 视野（view）、投影（projection） 和 世界（world），你可以使用以下常量之一：

常量 描述
matrix_view 当前的视野矩阵
matrix_projection 当前的投影矩阵
matrix_world 当前的世界矩阵


语法：

matrix_get(type);


参数：

参数 描述
type 获取值的矩阵类型（参见上面列出的 常量）


返回：

Array (1D, 16 elements)（一维数组，16个元素）


举例：

v_array = matrix_get(matrix_view);

上面的代码将获取当前视野矩阵的值，并使用它们填充数组 “v_array”。
