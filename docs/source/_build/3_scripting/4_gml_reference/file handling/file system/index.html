文件系统函数

GameMaker Studio 2 中存在许多函数，使你能够以不同的方式操作文件：

  • file_exists
  • file_delete
  • file_rename
  • file_copy
  • file_find_first
  • file_find_next
  • file_find_close
  • file_attributes

以下函数可用于更改文件名：

注意： 这些函数不适用于实际文件！它们只处理与不同路径元素相关的字符串，应与其他文件函数一起使用。
  • filename_name
  • filename_path
  • filename_dir
  • filename_drive
  • filename_ext
  • filename_change_ext

最后，有两个特殊的函数用于获取文件名，这些函数可用于使用文件类型过滤器加载和保存文件：

注意： 这些函数 仅 Windows 有效
  • get_open_filename
  • get_open_filename_ext
  • get_save_filename
  • get_save_filename_ext
警告！ 由于 GameMaker Studio 2 被沙箱化，这些函数可能无法正常工作！有关详细信息，请参阅 文件系统限制 部分。
