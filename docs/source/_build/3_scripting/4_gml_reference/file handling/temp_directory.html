temp_directory

语法：

temp_directory


返回: String（字符串）。


描述

这可用于返回每次运行时为游戏创建的临时目录（根目录不包含最终的 “\”）。该目录将保存文件，并且可以在游戏运行时访问，但是当游戏关闭时，它将被删除（以及它包含的所有文件）。


警告！ 此函数可能无法正常工作！ 因为 GameMaker Studio 2 被沙盒化。有关更多信息，请参阅 文件系统限制 一节。


举例：

ini_open(temp_directory + "\temp_ini.ini");

这将在游戏的临时目录中打开一个 ini 文件（如果它尚不存在则创建它）。
