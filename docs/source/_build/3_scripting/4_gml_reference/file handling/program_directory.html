program_directory

语法：

program_directory


返回: String（字符串）。


描述

这将返回存储游戏可执行文件的目录。 然而，这可能并不总是有用，特别是因为一些设备从 *.zip 文件运行 exe，所以无论游戏实际在哪里运行，这都会返回相同的内容。


警告！ 此函数可能无法正常工作！ 因为 GameMaker Studio 2 被沙盒化。有关更多信息，请参阅 文件系统限制 一节。


举例：

dir = program_directory;

这将获取存储在 program_directory 中的可执行文件目录并储存在变量 “dir” 中。
