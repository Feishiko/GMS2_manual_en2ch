file_bin_open

Syntax:

file_bin_open(fname, mode);


参数 描述
fname 要读取的文件的名称。
mode 打开文件的模式（对文件的读写权限）。


返回： Real（实数）


描述

此函数将打开带有指定名称的二进制文件。 模式指示可以对文件执行的操作：

  • 0 = 只读
  • 1 = 只写
  • 2 = 可读可写

当文件不存在时，它将被创建，并且该函数返回必须在其他函数中使用的文件的 id。 你可以同时打开多个文件（最多 32 个），但一旦完成操作，请不要忘记关闭它们。

警告！ 此函数可能无法正常工作！ 因为 GameMaker Studio 2 被沙盒化。有关更多信息，请参阅 文件系统限制 一节。

注意：当目标模块是 HTML5 时，这些函数 不起 作用。


例如：

file = file_bin_open('myfile.bin', 2);

这将从与游戏相同的目录中打开一个文件，并将其索引分配给变量 “file”。
