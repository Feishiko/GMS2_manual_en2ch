file_bin_seek

Syntax:

file_bin_seek(binfile, pos);


参数 描述
binfile 要读取的文件的 ID。
pos 要移动到的文件中的位置。


返回： N/A


描述

此函数将文件中的当前读取位置移动到指定的位置。你提供由函数 file_bin_open 返回的文件 ID 值，在写入之前将位置移动到文件的大小，并追加（append）文件。

注意：当目标模块是 HTML5 时，这些函数 不起 作用。


例如：

file = file_bin_open('myfile.bin', 2);
size = file_bin_size(file);
file_bin_seek(file, size);

这将从与游戏相同的目录中打开一个文件，并将其索引分配给变量 “file”，然后获取文件的大小并将下一个写入位置设置为该大小。
