directory_create

语法：

directory_create(dname)


参数 描述
dname 要创建的目录名称。


返回： N/A（无返回值）


描述

此函数将在沙盒本地保存区域中创建指定名称的目录。


警告！ 此函数可能无法正常工作！ 因为 GameMaker Studio 2 被沙盒化。有关更多信息，请参阅 文件系统限制 一节。


举例：

if !directory_exists("Games")
   {
   directory_create("Games");
   }

这将检查本地数据文件夹中是否存在指定的目录，如果不存在，则会为你创建该目录。
