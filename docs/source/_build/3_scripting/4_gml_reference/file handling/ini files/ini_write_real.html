ini_write_real

语法：

ini_write_real(section, key, value);


参数 描述
section 写入 .ini 的键所在的节。
key .ini 要写入的键。
值 写入的实数值。


返回： N/A（无返回值）


描述

你可以使用此函数将值（数字）写入 ini 数据文件。Ini 文件由 节（sections） 组成，然后每个节由 键 - 值 对（key - value pairs）组成。所以典型的 ini 文件看起来像这样：


举例：

score = 1000;
ini_open( 'savedata.ini' );
ini_write_real( 'save1', 'Score', score );
score2 = ini_read_real( 'save1', 'Score', 0 );
ini_close();

这会将得分设置为 1000，然后打开 'savedata.ini' 并将此值写入 'save1' > 'Score'。然后，它将 score2 设置为其中的 'save1' > 'score' 键的实数值 1000，并再次关闭 .ini。如果没有找到此值，则 score2 最终会设置为 0。
