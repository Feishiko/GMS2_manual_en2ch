ini_section_exists

语法：

ini_section_exists(section);


参数 描述
section 打开 .ini 文件中要检查的节。


返回: 布尔值


描述

此函数检查当前打开的 ini 中是否存在某个节（section）。这不是防止错误的必要检查，因为当一个节不存在时，从不存在的节读取将只返回一个默认值，但是查看一个 ini 文件是否保存了特定数据会很有用。


举例：

ini_open("savedata.ini");
if !ini_section_exists("save1")
   {
   global.savegame = '1';
   }
ini_close();

如果找不到 'save1'，这会将变量 global.savegame 设置为 '1'。使用 for 循环，这可以用作计算游戏有多少个游戏存档的基本方式（即：为一个游戏创建多个同时保存游戏存档的系统）。
