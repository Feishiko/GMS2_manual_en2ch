ini_open_from_string


描述

此函数将从字符串创建一个 ini 文件并将其打开以进行读取 / 写入。字符串应正确格式化为 ini 文件（即：带有节、换行符和值），否则将无法正确创建 ini 文件。请注意，此 ini 文件是临时文件，将在关闭时从内存中删除，丢失存储在其中的任何信息，但 ini_close 函数将返回一个完整的 ini 文件字符串 ，然后可以保存到服务器或磁盘。


语法：

ini_open_from_string(string);


参数 描述
string 包含所有 ini 信息的字符串。


Returns:

N/A（无返回值）


例如：

ini_open_from_string(str)
global.sound = ini_read_string("Options", "Sound", true);
ini_close();

上面的代码设置了一个临时 ini 来保存字符串 “str” 中的信息，然后在再次关闭它之前从 ini 读取。
