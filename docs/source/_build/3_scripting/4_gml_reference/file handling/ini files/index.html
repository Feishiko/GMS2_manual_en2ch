Ini 文件


INI 文件是小而轻巧的文件，与大多数平台兼容。它们非常适合存储 小块 信息，如界面首选项、本地高分、关卡数据等......因此，你可以轻松方便地保存游戏中的信息。但是，值得注意的是，它们对可以存储的数据有限制，最多可以存储 64 KB，因此如果要保存数据结构等内容，你可能希望查看其他一些文件函数，例如 文本文件。

请注意，对于具有本地化的游戏，从外部文件中读取重音字母非常重要。当之前没有提供文件时，你应 首先 以 UTF8 格式创建 ini 文件（而不是用 GameMaker Studio 2 文件函数创建的默认 ANSI 格式 ini 文件），然后将其作为内含文件添加到 GameMaker Studio 2 中，以便在运行游戏时导出并使用。通过这种方式，你可以正确地读取和写入所有重音和非罗马字母。

对于 ini 文件存在以下函数：

  • ini_open
  • ini_close
  • ini_write_real
  • ini_write_string
  • ini_read_real
  • ini_read_string
  • ini_key_exists
  • ini_section_exists
  • ini_key_delete
  • ini_section_delete
  • ini_open_from_string
警告！ 由于 GameMaker Studio 2 被沙箱化，这些函数可能无法正常工作！有关详细信息，请参阅 文件系统限制 部分。
