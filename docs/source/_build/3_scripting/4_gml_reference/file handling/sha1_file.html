sha1_file

Syntax:

sha1_file(filename)


参数 描述
filename 用于生成 sha1 散列的文件。


返回： String（字符串）


描述

在密码学中，SHA-1 是由美国 国家安全局 设计的加密散列函数，用于几种广泛使用的流行 Git 应用程序中检查文件更改，还用于 TLS、SSL、PGP、SSH、S/MIME 和 IPsec 协议。此函数将获取输入文件，并返回该文件所特有的 ASCII 格式的 160 位消息摘要，以便在日后进行完整性验证。


例如：

hash = sha1_file(working_directory + "\game_data.ini")

上面的代码将为指定的文件生成 sha1 散列，并将返回的值存储在变量 “hash” 中。
