ds_grid_add_disk


描述

这个函数可以被用于向一个栅格中一个确定圆盘区域内所有单元格的数值中叠加一个被给定的值（实数或字符串）。这个待叠加的值必须和这些栅格的单元格中所承载的值是同一类型，比如：你不能把一个字符串叠加给一个实数，反之亦然，并且对于字符串来说这相当于串接


语法：

ds_grid_add_disk(index, xm, ym, r, val);

参数 描述
index 栅格的索引
xm 栅格内圆盘区域x位置。
ym 栅格内圆盘区域的y位置。
r 栅格中圆盘的半径
val 待叠加给圆盘内单元格的值


Returns:

N/A（无返回值）


例如：

ds_grid_add_disk(grid, 7, 6, 5, 2)

这将会向索引为变量"grid"的栅格结构中确定圆盘区域内的单元格中的所有值叠加2。
