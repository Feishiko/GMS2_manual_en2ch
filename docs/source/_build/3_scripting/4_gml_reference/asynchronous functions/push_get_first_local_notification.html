push_get_first_local_notification


描述

这个函数将以一系列键值对儿（key/value pairs）的形式返回一个 ds_map作为本地通知队列中最先被显示的通知。返回 -1 代表通知队列为空即没有通知，返回其他的真值为通知的ID。使用这个ID可以通过调用push_cancel_local_notification()来取消通知。

ds_map 将包含以下的键值对儿（key/value pairs）：

  • "title" － 通知的标题 ( 仅在Android 平台可用 ， iOS 平台不保存标题信息)
  • "message" － 消息正文
  • "data" － 字符串格式的数据包
注意： This function is limited to the iOS and Android target modules.


语法：

push_get_first_local_notification(map);

参数 描述
map ds_map 的ID 。


返回：

实数


例子：

var map = ds_map_create();
var ntf = push_get_first_local_notification(map);
while(ntf >= 0)
   {
   var data = ds_map_find_value(map, "data");
   if data == "Daily_Reward"
      {
      push_cancel_local_notification(ntf);
      }
   ntf = push_get_next_local_notification(map);
   }
ds_map_destroy(map);

上面的代码创建了一个 ds_map ，然后请求数据队列中第一个可用的通知。如果通知存在， map 将被赋值，通过代码查找key为 "data"的特定字符串。如果找到该字符串,则通知取消,继续检查队列中的下一个通知,直到所有通知都被检查。
