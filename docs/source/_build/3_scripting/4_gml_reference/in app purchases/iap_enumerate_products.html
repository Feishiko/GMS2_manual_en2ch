iap_enumerate_products

使用目标存储中当前可用的所有 IAP 产品填充 ds_list。

语法：

iap_enumerate_products();


返回： N/A（无返回值）


描述

此函数检索遇到的应用内购产品 ID 列表。在大多数平台上这只会返回通过 iap_activate() 提供的一套产品 id，但在存储平台上提供了设施检索可用产品的完整列表标题那么这组产品 id 将包括任何和所有可用的产品 id。该函数要求你提供一个以前创建的ds_list（如果它被用于其他目的，你应该首先清除它），该列表将由函数调用填充。无论使用何种平台，如果目标商店不可用，那么当你使用 iap_activate 激活购买时，它将从提供的商店获得产品 id。


例如：

var status = iap_status();
var iap_list = ds_list_create();
if status == iap_status_available
   {
   iap_enumerate_products(iap_list);
   }

上面的代码检查目标 IAP 存储的状态，如果可用，它将用可用的产品 id 填充创建的列表，以备将来使用。
