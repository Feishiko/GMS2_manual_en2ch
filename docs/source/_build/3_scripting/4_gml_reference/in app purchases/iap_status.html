iap_status

允许你跟踪应用内购买系统的状态。

语法：

iap_status();


返回： 常量


描述

此函数允许你跟踪 GameMaker Studio 2 应用内购系统的状态，并返回以下常量之一：

常量 描述
iap_status_uninitialised 应用内购尚未激活你的游戏。
iap_status_unavailable 目标商店没有可用的连接。
iap_status_loading 与目标商店的连接退出并正在激活。
iap_status_available 购买已激活且目标商店可用。
iap_status_processing 目前游戏与目标商店之间存在通信，例如进行购买或恢复购买等……
iap_status_restoring 在 iOS 目标上，当你的游戏试图恢复购买时，可以返回该值。


例如：

status = iap_status();

上述代码请求当前游戏的商店状态，并将其存储在变量中以供检查。
