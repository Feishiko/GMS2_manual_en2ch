camera_set_default


描述

此函数可以用将你之前使用 camera_create 函数创建出来的摄像机设为默认摄像机。如果你 没有 为游戏创建可用的视野摄像机，GameMaker Studio 2 仍将使用一个摄像机来显示游戏中的动作。这个摄像机就是 默认 摄像机，可以进行和其它摄像机一样的操作（包括销毁）。


语法：

camera_set_default(camera_id)


参数：

参数 描述
camera_id 创建相机时返回的唯一相机 id 值


Returns:

N/A（无返回值）


例如：

var new_cam = camera_create_view(0, 0, 640, 480, 0, obj_Player, 5, 5, -1, -1); camera_set_default(new_cam);

上述代码创建一个新摄像机，并将它设为默认摄像机。
