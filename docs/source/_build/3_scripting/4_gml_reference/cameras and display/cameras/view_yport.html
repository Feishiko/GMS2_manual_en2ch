view_yport

描述

通过此内置数组，你可以获取或设置给定可视区域的 y 位置。可视区域是屏幕上视野被绘制的区域，你最多能同时拥有八个活动的可视区域（数组内的编号是0到7）。现在， GameMaker: Studio默认游戏窗口（或者背景画布）和房间一样大，然而当你激活可视区域和摄像机时，游戏窗口的大小将是所有可视区域的覆盖的总和 。所以，如果在两个位置由两个不同的可视区域，覆盖的总面积定义游戏窗口的大小。如下图说明：

如果你只有一个可视区域，并设置它的x或y值，会产生一些有趣的效果，但通常不实用（见下图），因此最好将x和y位置保持为（0,0）。



语法：

view_yport[0 .. 7];


Returns:

实数


例如：

view_xport[0] = 0;
view_yport[0] = 0;

上述代码将重设view port[0]（可视区域）的位置为 显示器的(0,0)位置 (左上角)。
