camera_get_view_target


描述

此功能可用于返回给定摄像机的跟随目标。返回值可以是对象索引，实例ID。如果没有设置跟随目标，则返回—1。


语法：

camera_get_view_target(camera_id)


参数：

参数 描述
camera_id 创建相机时返回的唯一相机 id 值


Returns:

对象索引，实例ID，如没有目标则返回-1。


例如：

var target = camera_get_view_target(view_camera[0]);
if target != obj_Player
   {
   camera_set_view_target(view_camera[0], obj_Player);
   }

上述代码检查被分配给view port[0]的相机所跟随的目标，然后检查它是否是“obj_target"。如果不是的话，那就让此视野摄像机跟随那个对象的一个实例。
