view_xport

描述

With this built in array you can get or set the x position of the given view port. The view port is the area on the screen where the view is drawn, and you can have up to 8 active at any one time (the array is values from 0 to 7 inclusive to give 8 ports). Now, the default for GameMaker Studio 2 is that the game window (or background canvas) is the same size as the room, however when you activate view ports and cameras, this behaviour changes and the total size of the bounding box for all view ports is used. 所以，如果有两个不同的视口在两个不同的位置，覆盖的总面积定义游戏窗口的大小。如下图说明：

If you only have one view port setting the x or y value of the port can have some interesting effects but is not normally practical (see the image below) and so it is better to maintain the x and y position as (0,0).



语法：

view_xport[0 .. 7];


返回：

实数


举例：

view_xport[0] = 0;
view_yport[0] = 0;

上述代码将重设view port[0]（视口）的位置为 显示器的(0,0)位置 (左上角)。
