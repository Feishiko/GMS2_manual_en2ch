view_wport


描述

这个变量可以用来获取或设置特定可视区域的宽度。可视区域的宽度（如果有多个活动的窗口，将是组合可视区域的宽度），决定了游戏开始时游戏窗口或背景画布的宽度，因此在游戏开始后更改此值 将不会改变窗口尺寸，除非使用GML函数 window_set_size。如果可视区域比分配的视野更大或更小，视野将被缩放以适应可视区域大小，如下图所示：


语法：

view_wport[0 ... 7];


Returns:

整数


例如：

camera_set_view_size(view_camera[0], view_wport[0], view_hport[0]);

上述代码把摄像机视野的高度和宽度设为和可视区域的一样。
