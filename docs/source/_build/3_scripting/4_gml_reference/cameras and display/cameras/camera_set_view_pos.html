camera_set_view_pos


描述

你可以用此函数来更新房间内的摄像机的位置。你需要提供摄像机的独特ID（被另外的camera_create 函数所创建），并给出需要将摄像机设置的位置的x和y坐标。


语法：

camera_set_view_pos(camera_id, x, y)


参数：

参数 描述
camera_id 当你创建摄像机时的独特ID值
x 设置视野的X位置（在房间内）
y 设置视野的Y位置（在房间内）


Returns:

N/A（无返回值）


例如：

camera_set_view_pos(view_camera[0], x - (view_wport[0] / 2), y - (view_hport[0] / 2));

上述代码将会设置被分配给view port[0]的摄像机的位置。
