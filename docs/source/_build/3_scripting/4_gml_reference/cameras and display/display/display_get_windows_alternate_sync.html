display_get_windows_alternate_sync

获取当前的垂直同步方法（只适用于windows）。

语法：

display_get_windows_alternate_sync();


返回: 布尔值


描述

这个只用于 Windows 的函数在运行时获取当前垂直同步的方法。如果你激活了垂直同步的替代方法（断水注：并非返回垂直同步是否激活，而是指替代方法），它将返回true ，或者如果你使用常规的垂直同步方式，它则返回false 。注意这个函数并不是报告垂直同步是否激活（这个通过切换函数 display_reset()实现），你可以在全局游戏设定中或者或者使用函数 display_set_windows_alternate_sync()来设置垂直同步的方法。


举例：

if !display_get_windows_alternate_sync()
   {
   display_set_windows_alternate_sync(true);
   }

上述代码检测是否使用于替代的垂直同步方法，如果是默认的则切换为替代的方法。
