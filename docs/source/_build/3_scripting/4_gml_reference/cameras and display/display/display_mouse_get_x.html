display_mouse_get_x

返回鼠标相对于显示器的x坐标。

Syntax:

display_mouse_get_x();


返回： Real（实数）


描述

这个函数返回 屏幕上鼠标指针的x坐标。应该注意的是，这个函数只在windows平台有效。当然HTML5也可以一用，但只是返回相对于画布(0,0)的一个值，鼠标在画布外面时不会返回任何值。For other devices it will return 0, and you should use the device_mouse_raw_x and device_mouse_raw_y functions instead.


例如：

my_x = display_mouse_get_x();

上述将设置 my_x 等于相对于显示器的鼠标x坐标。


返回: 窗口和视野
