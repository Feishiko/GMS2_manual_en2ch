display_get_windows_vertex_buffer_method

获取 Windows 目标平台的缓冲区方法。

语法：

display_get_windows_vertex_buffer_method();


返回:常量


描述

这个只用于 Windows 的函数，允许你找出当前处理游戏几何顶点缓冲区的方法。这个可以在全局游戏设定中或者使用函数 display_set_windows_vertex_buffer_method()设置，然后返回下面的一个常量：

常量 描述
vbm_fastvbm_fast 这是默认的，且是Nvidia推荐的方法 - 它应该是最高的性能。
vbm_compatible This is the deprecated method that GameMaker Studio 2 used to use before adopting the fast method - poorer performance on newer machines
vbm_most_compatible 这是 GameMaker 之前使用的一个传统方法 - 性能不如上面两个，但是应该兼容几乎任何一台机器。


举例：

if display_get_windows_vertex_buffer_method() != vbm_fast
   {
   display_set_windows_vertex_buffer_method(vbm_fast);
   }

上述代码将检测当前的几何顶点缓冲区的方法，如果没有准备好则设置为 “fast（vbm_fast 方法）”。
