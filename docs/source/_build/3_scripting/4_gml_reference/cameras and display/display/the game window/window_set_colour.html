window_set_colour

设置游戏窗口内但是视野之外区域的颜色（默认黑色）。

语法：

window_set_colour(colour);


参数 描述
colour 颜色，用于设置那块区域。


返回：N/A


描述

这个函数可以设置游戏窗口的背景颜色。这个颜色表示的游戏窗口内视野间隙的背景颜色。下面的图片将说明这一点：

上面的图片输出了两个视野，每一个绘制在不同的位置。拉伸游戏窗口容纳两个视野输出，并用窗口颜色填充没有视野画面的区域。


例如：

if window_get_colour() != c_black
   {
   window_set_colour(c_black);
   }

上述代码检测窗口内背景颜色，如果不是黑色则设置为黑色。
