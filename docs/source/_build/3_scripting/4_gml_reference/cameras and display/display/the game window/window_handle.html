window_handle

返回游戏窗口的id。

语法：

window_handle();


返回： 指针


描述

使用此函数你可以得到 <<span class="notranslate">窗口ID值（HWND，即窗口句柄，一个指针）此函数对那些需要在DLL代码需要window handle 调用 Windows API's （此返回的指针被转换为一个字符串，如果在c++中。只转换成一个HWND）。下表显示了支持的平台及其返回的内容：

平台 返回
Windows Window HWND
macOS NSWindow类
Ubuntu (Linux) XWindow handle
HTML5 Canvas ID


例如：

win_id = windows_handle();

上述代码将游戏窗口id存入一个变量。
