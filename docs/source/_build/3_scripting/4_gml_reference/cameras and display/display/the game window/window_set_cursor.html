window_set_cursor

设置游戏窗口的普通光标。

语法：

window_set_cursor(cursor);


参数 描述
cursor 为游戏窗口设置的光标。.


返回：N/A


描述

有了这个函数，你可以设置游戏窗口的光标为下面列出的任意一个常量（获取当前使用的光标可以使用函数window_get_cursor ，它将返回这些常量中的一个）：

常量 Cursor
cr_none

cr_default
cr_arrow
cr_cross
cr_beam
cr_size_nesw
cr_size_ns
cr_size_nwse
cr_size_we
cr_uparrow
cr_hourglass
cr_drag
cr_appstart
cr_handpoint
cr_size_all


例如：

if mouse_check_button_pressed(mb_left)
   {
   window_set_cursor(cr_drag);
   }

上述代码在鼠标左键按下时改变窗口光标为拖动光标。
