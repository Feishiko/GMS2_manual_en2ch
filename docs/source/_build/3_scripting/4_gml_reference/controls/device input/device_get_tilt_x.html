device_get_tilt_x

此函数返回设备 x 轴上的倾斜量。

语法：

device_get_tilt_x()


返回： Real（实数）


描述

此函数返回介于 -1 和 1 之间的值，具体取决于设备的 “倾斜” 角度。倾斜度和返回值之间的实际相关性取决于它使用的设备和操作系统，但一般情况下值 1 或 -1 与 +/- 90°相同。下图显示了每个可用函数与设备的关系：


例如：

if display_get_orientation() = display_landscape
   {
   y += sign(device_get_tilt_y());
   }
else
   {
   y += sign(device_get_tilt_x());
   }

上面的代码检查显示器的方向，然后使用相应的倾斜值沿 y 轴移动玩家。
