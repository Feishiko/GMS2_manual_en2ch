device_mouse_dbclick_enable

设置是否应将双击左键检测为 mb_right。

语法：

device_mouse_dbclick_enable(bool);


参数 描述
bool 设置打开双击检测（true）或关闭（false）。


返回: 布尔值


描述

此函数可用于将设备设置为检测 mb_left（鼠标左键）的双击是否为 mb_right（鼠标右键）的单击。默认设置为 true，这意味着每次用户快速连续点击设备屏幕两次时，返回值与单击鼠标右键相同。启用此选项后，第一次点击将被检测为 mb_left，第二次点击将被检测为 mb_right，因此请确保你使用的任何代码都将此考虑在内。


例如：

if os_type == os_windows || os_type == os_mac
   {
   device_mouse_dbclick_enable(false);
   }

上面的代码检查运行游戏的设备是 Windows PC 还是 Mac，如果它们是其中之一，它会禁用双击功能。
