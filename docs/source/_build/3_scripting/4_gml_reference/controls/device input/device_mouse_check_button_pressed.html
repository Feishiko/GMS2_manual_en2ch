device_mouse_check_button_pressed

返回是否按下了给定的设备按钮。

语法：

device_mouse_check_button_pressed(device, button);


参数 描述
device 正在检查的设备（从 0 - n）。
button 正在检查的设备的按钮。


返回: 布尔值


描述

此函数返回 true 或 false，具体取决于你指定的设备是否已被 “触摸”（单击）。device 参数是指触摸编号，可以是 0 到 n（可以检测到的最大触摸次数在很大程度上取决于所使用的设备和它运行的操作系统）。此函数仅在实际按下操作时触发 一次，并且 mb_left 和 mb_right 的标准常量可用于检查任一按钮。

例如：

if device_mouse_check_button_pressed(0, mb_left)
{
press=true;
}

上面的代码检查是否已按下设备并将变量设置为 true（如果有）。
