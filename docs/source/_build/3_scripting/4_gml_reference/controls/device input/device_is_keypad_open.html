device_is_keypad_open

检查设备以查看键盘是否打开。

语法：

device_is_keypad_open


返回: 布尔值


描述

这会检查键盘的设备，如果有可用，则返回 true，否则返回 false。请注意，此功能主要用于 Android 设备。

对于那些使用 Sony Xperia Play 的用户，在 GameMaker Studio 2 中有一个设置按钮/按键映射设置，因此对于游戏手柄键你可以使用以下 键盘常量 ：vk_up，vk_down，vk_left，vk_right，而 Triangle 是 ord("T")，Square 是 ord("S")，Circle 是 vk_alt + vk_backspace，而 back 按钮只是 vk_backspace，Cross 是 vk_space，Select 是 vk_return 且 Start 是 vk_rshift，L Trigger 是 ord("L") 和 R Trigger 是 ord("R")</ t29>。


例如：

if device_is_keypad_open()
   {
   global.Setting = 2;
   }
else
   {
   global.Setting = 1;
   }

上面的代码检查键盘然后根据返回的值更改全局变量。
