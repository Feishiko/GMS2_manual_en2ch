gamepad_set_colour


描述

此函数可用于设置 PlayStation 控制器内 LED 的颜色。指定要设置的设备插槽，然后提供一种颜色，可以是任何 颜色常数 或使用特定颜色函数或 HEX 值创建的颜色值（如 $FFFFFFF）。


语法：

gamepad_set_colour(device, colour);

参数 描述
device 要设置哪个游戏手柄设备 “插槽” 。
colour 要使用的颜色


Returns:

N/A（无返回值）


例如：

if health < 10
   {
   gamepad_set_colour(0, c_red);
   }

如果 health 变量低于 10，上面的代码会将 PlayStation 控制器 LED 设置为红色。
