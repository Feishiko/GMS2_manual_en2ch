mouse_check_button_pressed


描述

如果已按下正在检查的鼠标按钮，则此函数将返回 true；如果没有，则返回 false。对于任何鼠标按钮，只有在第一次按下时，该函数才会被触发一次，再次触发该按钮需要释放按钮并再次按下。请注意，它将被视为在步持续时间内触发，并且对于具有任何鼠标事件或使用此相同函数的所有实例。

你提供鼠标按钮以检查以下常量之一：

常量 描述
mb_left 鼠标左键
mb_middle 鼠标中键（这并非对所有目标平台都有效）
mb_right 鼠标右键
mb_none 无鼠标键
mb_any 任意鼠标键


语法：

mouse_check_button_pressed(numb);


参数 描述
numb 要检查的鼠标按键常量


返回：

Boolean（布尔值）


例如：

if mouse_check_button_pressed(mb_left)
   {
   score += 50;
   }

上面的代码将检查是否按下了鼠标左键，如果有，它将增加 50 分。
