gesture_rotate_angle


描述

此函数用于设置一对触摸必须超过的角度，以触发 旋转开始手势。角度以度为单位，默认值为 5°。


语法：

gesture_rotate_angle();


Returns:

实数（英寸）


例如：

if gesture_get_rotate_angle() != 5
    {
    gesture_rotate_angle(5);
    }

上面的代码检查手势的旋转角度是否设置为 5°，如果不是，则将其设置为该值。
