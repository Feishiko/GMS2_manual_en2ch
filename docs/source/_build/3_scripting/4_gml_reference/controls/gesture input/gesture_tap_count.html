gesture_tap_count


描述

此函数用于设置是启用还是禁用点击计数。当设置为 true 则点击计数启用，当设置为 false时，它将被禁用，尽管你可以使用函数 gesture_get_tap_count 检查它处于哪种状态。启用后，每个点击事件都会有一个额外的 "tapcount" 数据结构映射（ds map）条目，该条目会记录点击次数，启用后表示将触发 所有 点击事件，即：两次点击触发单击事件和双击事件，单击事件点击计数为 1，双击事件点击计数为 2。在为双击检测设置的时间过后，点击计数值将重置为 0（参见函数 gesture_double_tap_time）。如果禁用了点击计数，则在双击时间过去且未检测到第二次点击之前，不会注册初始点击。请注意，默认情况下，点击次数 已启用。


语法：

gesture_tap_count(enable);

参数 描述
enable 启用（设置为 true）或禁用（设置为 false）点击计数。


Returns:

N/A（无返回值）


例如：

if !gesture_get_tap_count()
    {
    getsure_tap_count(true);
    }

上面的代码检查是否启用了点击计数，如果没有，则启用它。
