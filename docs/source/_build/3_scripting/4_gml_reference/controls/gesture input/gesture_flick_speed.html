gesture_flick_speed


描述

此函数用于设置释放触摸或点击时触发 手势轻抚 事件所需的速度。速度以英寸/秒为单位测量，默认值为 2.0。


语法：

gesture_flick_speed(speed);


参数 描述
speed 为轻抚手势事件检测设置的速度（以英寸/秒为单位）。


Returns:

N/A（无返回值）


例如：

if gesture_get_flick_speed() != 2
    {
    gesture_flick_speed(2);
    }

上面的代码检查手势的轻抚速度是否设置为2 英寸/秒，如果不是，则将其设置为该值。
