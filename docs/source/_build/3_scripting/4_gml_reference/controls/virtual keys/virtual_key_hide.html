virtual_key_hide


描述

为设备创建虚拟键后，可能需要调试并测试它是否正确定位并正常工作，为此你可以调用函数 virtual_key_show。但是，你可能需要打开和关闭此视图，因此你还可以使用此函数隐藏虚拟键，使用要隐藏的虚拟键的索引（之前使用 virtual_key_add 创建和存储的索引）。一旦你对（虚拟键的位置和其正常工作）这件事感到满意，建议你在其区域中创建并放置自己的图形。


语法：

virtual_key_hide(index);

参数 描述
index 要隐藏的虚拟键索引。


返回：

N/A（无返回值）


例如：

if global.Debug
   {
   virtual_key_show(global.Left);
   }
else
   {
   virtual_key_hide(global.Left);
   }

上面的代码检查全局变量 “global.Debug”，如果它测试为 true，那么在变量 “global.Left” 中索引的虚拟键将在屏幕上绘制，如果为 false，则按键将被隐藏。
