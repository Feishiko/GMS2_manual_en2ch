keyboard_check_released


描述

使用此函数，你可以检查是否已释放了某个键。 与keyboard_check 函数不同，此函数仅在每次抬起键时运行一次，因此要再次触发，必须先按下键然后再次释放。该函数将获取函数 ord()（仅来自 A - Z 的 大写 字母或 0 - 9 中的数字）返回的键码值， 或主 键盘输入 页面上列出的任何 vk_* 常量。


语法：

keyboard_check_released(key);

参数 描述
key 检查释放状态的键。


Returns:

Boolean


例如：

if keyboard_check_released(ord("P"))
   {
   instance_create_layer(0, 0, "Controllers", obj_Pause);
   }

上面的代码将检查 “P” 键是否已被释放，如果是，则创建一个 “obj_Pause” 的实例。
