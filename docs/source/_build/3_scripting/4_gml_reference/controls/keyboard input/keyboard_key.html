keyboard_key


描述

使用此变量，你可以获取当前正在按下的键的键码，如果在完成检查时没有按下键，它将返回 0。


语法：

keyboard_key;


Returns:

Real（实数）


例如：

switch (keyboard_key)
   {
   case vk_numpad1: gun = weapon[0,0]; break;
   case vk_numpad2: gun = weapon[1,0]; break;
   case vk_numpad3: gun = weapon[2,0]; break;
   }

上面的代码使用 keyboard_key 变量的值将变量设置为与数组相同的值。
