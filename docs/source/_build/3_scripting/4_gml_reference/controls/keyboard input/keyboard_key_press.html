keyboard_key_press


描述

使用此函数，你可以模拟键盘上任何键的按下。使用时，按键将被标记为被按下，直到调用相应的释放函数（参见 keyboard_key_release()）。该函数接受任何 vk_ 常量以及标准键和 ascii 码。请注意，在调用此函数后，如果在键盘上按下 物理 按键，此按键事件将 不再 再次注册，直到该键已被物理释放（触发释放） 事件并停止此函数），或已调用相应的释放键函数，并再次按下该键。


语法：

keyboard_key_press(key);

参数 描述
key 模拟按下的键。


Returns:

N/A（无返回值）


例如：

keyboard_key_press(vk_space);

这将模拟空格键按下。
