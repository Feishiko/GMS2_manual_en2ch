event_number

返回当前所执行的事件编号。

语法：

event_number;


返回： Real（实数）


描述

此 只读 变量返回当前正在调用的事件的编号，其中该编号实际上是指事件的 “子事件”，即：对于步事件，事件编号可以是 ev_step_normal、ev_step_begin 或 ev_step_end 中的任何一个常数。有关特定子事件可用的常量的完整列表，请参阅 event_perform，如果你需要知道主事件本身，则应该检查 event_type。


例如：

switch (event_number)
   {
   case ev_step_normal: show_debug_message("Step event!"); break;
   case ev_game_start: show_debug_message("Game Start""); break;
   case ev_room_start: show_debug_message("Room Start!"); break;
   }

上面的代码可以通过脚本调用，用于显示调试消息，通知你当前正在触发哪个事件。
