object_get_visible

返回物体是否标记为可见。

语法：

object_get_visible(obj);


参数 描述
obj 待检测的物体索引。


返回： 布尔值


描述

此函数将告诉你正在检查的物体是否已标记为“可见”（运行其绘制事件）或为“不可见”（不运行其绘制事件）。请注意，这不是实例函数！因此，你可以拥有可见物体和同一物体的不可见实例，反之亦然。你可以使用 visible 实例变量设置单个实例可见性。


例如：

if !visible && object_get_visible(object_index) visible=true;

上面的代码将检查运行它的实例以查看它是否可见，以及检查实例的物体索引以查看它是否被标记为可见。如果实例 不 可见但物体索引标记为 on，则它将为该实例将 “visible（可见）” 设置为 true。
