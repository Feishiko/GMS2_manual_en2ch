object_set_visible

设置给定物体的可见。

语法：

object_set_visible( index, vis );


参数 描述
index 要更改的物体的索引。
vis 物体的新的可见值（true = 可见，false = 不可见）。


返回： N/A（无返回值）


描述

使用此函数，你可以将特定物体标记为可见或不是（有关 “可见” 标志的更多信息，请参阅 定义物体（Defining Objects））。这意味着 在可见更改后 创建的物体的所有实例都将使用这个新状态创建，而已经在房间中的实例可能不会受到影响。

请注意，这不是实例函数！你可以使用 persistent 变量设置各个实例的可见标志，因此有十个实例都标记为可见，即使在 IDE 里相关物体的标志设置为false，即使使用此函数将物体的可见标志更改为 true，当前房间中的所有实例都将可见原样，并且只有在调用该函数后创建的实例才会以可见标记为 true 开始。


例如：

if !object_get_visible(obj_Block) object_set_visible(obj_Block, true);

上面的代码检查物体 “obj_Block” 的可见标志，如果为 false，则将其更改为 true。
