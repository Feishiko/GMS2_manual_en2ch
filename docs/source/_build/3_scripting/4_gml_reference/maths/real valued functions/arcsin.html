arcsin


描述

返回x的反正弦。如果sin（val）= n，arcsin（n）= val。即val的余弦为n，那么n的反余弦将会是val。返回的数将在pi和0之间。

注意此值只能是在-1和1之间（其它的值将会抛出异常）。


语法：

arcsin(x);


参数 描述
x 返回的反正弦的角度（以弧度表示）。


Returns:

实数


例如：

val = arcsin( 0 );

这会将val设为0。
