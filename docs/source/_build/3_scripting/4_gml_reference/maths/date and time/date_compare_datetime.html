date_compare_datetime

Returns which of two times on specific dates is the latest.

语法：

date_compare_datetime(date1, date2);


参数 描述
date1 第一个日期时间。
date2 第二个日期时间。


返回： Real（实数）


描述

使用此函数来检查两个日期和时间，看哪个更早，哪个更晚。如果日期时间1更早，返回-1，如果两者相同，返回0，如果日期时间2更早，返回1。返回0的情况是非常少见的，因为必须是同年同月同日同时同分同秒。


例如：

d = date_compare_datetime(date_create_datetime( 2011, 9, 15, 11, 4, 0 ), date_current_datetime());

此代码将会根据两个日期的早晚来为"d"赋值，很显然将会赋1，因为当前日期时间要比2011年9月15日晚很多。
