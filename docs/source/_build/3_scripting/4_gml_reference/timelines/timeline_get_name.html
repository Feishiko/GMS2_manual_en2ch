timeline_get_name

获取给定时间轴的名称。

语法：

timeline_get_name( ind );


参数 描述
ind 要检查的时间轴索引的名称。


返回： String（字符串）


描述

此函数可用于以字符串形式获取时间轴的名称。如果使用 timeline_add 函数动态创建了时间轴，则返回的名称的格式为 “__newtimelineN”，其中 “N” 是时间轴的编号（从 0 开始）。请注意，这是 仅 一个字符串，不能用于直接引用时间轴 - 为此，你将需要 时间轴索引。但是，可以使用这个字符串和函数 asset_get_index() 一起使用返回的字符串获得 时间轴索引。


例如：

str = timeline_get_name(global.tl);

上面的代码将存储索引在变量 “global.tl” 中的时间轴的名称。
