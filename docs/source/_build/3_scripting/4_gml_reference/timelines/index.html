时间轴

时间轴是控制游戏中事件的强大机制，它基于 “时刻”，而 “时刻” 只是游戏中的单个步。这些时间轴通常使用 时间轴编辑器（Timeline Editor） 以与物体类似的方式从 资源树（Resource Tree）创建，即：你创建时间轴，然后在你需要执行操作的每个时刻 ，添加一些代码或 DnD™ 图标使得它们在该时刻被执行。创建后，你可以通过代码停止、启动和操作时间轴。你甚至可以通过定义脚本并从控制器物体（例如）动态地将其添加到时间轴中来修改单个时刻，从而使其成为创建游戏时非常强大且功能多样的工具。

以下函数用于在游戏中添加和操作时间轴：

  • timeline_exists
  • timeline_get_name
  • timeline_add
  • timeline_delete
  • timeline_moment_add_script
  • timeline_moment_clear
  • timeline_clear
  • timeline_size
  • timeline_max_moment

以下变量 “内置” 到所有实例中，以便更轻松地使用时间轴并控制它们：

  • timeline_index
  • timeline_running
  • timeline_speed
  • timeline_position
  • timeline_loop
