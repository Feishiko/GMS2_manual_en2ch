timeline_delete

删除给定的时间轴。

语法：

timeline_delete( ind );


参数 描述
ind 要删除的时间轴索引。


返回： N/A（无返回值）


描述

有了这个函数，你可以从你的游戏删除任何时间轴。如果这条时间轴是在 GameMaker Studio 2 资源树中创建的，请注意它已被完全从游戏中删除，使用此函数后试图再次调用索引时间轴将导致错误。如果时间轴是使用 timeline_add 函数动态创建的，那么应该在不再需要时间轴的时候使用该函数，以防止任何可能会减慢（并最终导致游戏崩溃）的内存泄漏。

重要！该函数在试用版（Trial License）产品中不可用。


例如：

if timeline_exists(global.tl) timeline_delete(global.tl);

上面的代码检查时间轴是否存在，并在全局变量 “tl” 中建立索引，如果存在，则删除该时间轴。
