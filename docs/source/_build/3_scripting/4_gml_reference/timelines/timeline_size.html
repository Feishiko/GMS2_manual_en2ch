timeline_size

返回给定时间轴中的活动时刻数。

语法：

timeline_size(ind);


参数 描述
ind 时间轴的索引从中获取信息。


返回： N/A（无返回值）


描述

使用此功能，你可以获得时间轴的活动时刻总数（“活动” 时刻是添加了代码或 DnD™ 的时刻）。使用 timeline_moment_add_script() 和 timeline_moment_clear() 函数创建动态时间轴时，这非常方便，你可以检查给定时间轴是否具有正确的活动时刻数或根本没有活动时刻。


例如：

if timeline_size(global.tl) == 0
   {
   timeline_moment_add_script(global.tl, room_speed + irandom(room_speed), scr_spawn_enemy);
   }

上面的代码检查给定的时间轴大小，如果它返回 0（即：时间轴没有活动时刻），它将添加一个脚本，用于在全局变量 “tl” 中索引的时间轴内的随机时刻。
