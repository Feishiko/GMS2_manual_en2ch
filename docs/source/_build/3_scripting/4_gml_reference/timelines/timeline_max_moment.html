timeline_max_moment

为你提供给定时间轴中的最大时刻。

语法：

timeline_max_moment(ind);


参数 描述
ind 获取最后一刻的时间轴索引。


返回： N/A（无返回值）


描述

此函数将返回时间轴执行其最终操作的时刻的值。因此，如果你有一个时间轴，其中 8 个不同的动作相隔 20 个时刻，则此函数将返回该时间轴中最后一个动作的时刻值，即 160。此函数是测试时间轴在运行时是否已经过最后一个活动时刻的好方法，因为时间轴位置将在游戏的每步继续递增，无论是否有进一步的操作。


例如：

if timeline_position > timeline_max_moment(timeline_index)
   {
   timeline_running = false;
   }

上面的代码将根据最大活动时刻检查当前时间轴位置，如果它更大，则将停止时间轴。
