string_insert


描述

使用这个函数，你可以创建一个由两个字符串组成的新字符串，其中一个字符串在给定的位置被插入到另一个字符串中。例如，将用户名添加到预定义的文本中，从而让游戏的玩家感觉更参与到操作中，这是非常有用的。


语法：

string_insert(substr, str, index);

参数 描述
substr 要插入的子字符串。
str 要复制的字符串。
index 插入子字符串的字符串中字符的位置。


返回：

String（字符串）


例如：

str2 = string_insert(username, "Hello, , how are you?", 8);

这将在给定的短语中插入 “username” 变量中的字符串，结果字符串如下所示：“Hello, NAME, how are you?”。
