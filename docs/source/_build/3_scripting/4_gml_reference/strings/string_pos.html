string_pos


描述

这个函数将返回子字符串实例在字符串中的字符位置，如果没有找到，则返回 0。这样做的一个用途是过滤可能被认为具有冒犯性的单词，或者找到将一些文本插入另一个字符串的正确位置。


语法：

string_pos(substr, str);

参数 描述
substr 要在字符串中查找的子字符串。
str 被查找的字符串


返回：

Real（实数）


例如：

if string_pos(",", text) != 0
   {
   string_insert(name, text, string_pos(",", text));
   }

上面的代码在字符串 “text” 中搜索一个逗号，如果找到一个逗号，就在那个位置插入子字符串 “name”。
