ord


描述

此函数接受输入字符串的单个字符，并返回该字符的 Unicode（UTF8）值。请注意，当与 keyboard_check* 函数一起使用时，输入字符串的长度只能是一个字符，并且只能是 0 到 9 之间的数字或从 A 到 Z 的 大写罗马字符。


语法：

ord(string);

参数 描述
string 用于查找 Unicode 代码的字符串。


返回：

Real（实数）


例如：

if keyboard_check(ord("W"))
    {
    y -= 4;
    }

如果按住键 W，这将使调用代码的实例向上移动四个像素。
