string_hash_to_newline


描述

此函数返回一个字符串，其中 “#” 符号将转换为新行。所以字符串格式化后为：

"Hello#World"

打印到屏幕上为：

Hello
World

重要！这个函数是为了在 GameMaker Studio 2 和之前版本的软件之间导入兼容性而提供的，因此除非有必要复制过时的函数，否则不应该使用这个函数。相反，你应该使用 转义字符 来标记新行。有关导入兼容性的更多信息，请参阅 此处。


语法：

string_hash_to_newline(string);

参数 描述
string 要转换为多行的字符串


返回：

String（字符串）


例如：

var str = string_hash_to_newline("Hello#World");
draw_text(32, 32, str);

上面的代码将带有 # 符号的字符串转换为分成两行的字符串，然后将其打印到屏幕上。
