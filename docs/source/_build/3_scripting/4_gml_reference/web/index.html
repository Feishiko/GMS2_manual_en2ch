Web（网页端）


下面列出的函数是可以在所有目标平台上使用的常规 Web 函数：

  • url_open
  • url_open_ext
  • url_open_full
  • url_get_domain
  • http_request
  • http_get
  • http_get_file
  • http_post_string

除了上面列出的一般函数外， HTML5 目标 还有一些额外的网络函数，专门设计用于以它为目标的游戏：

  • browser_height
  • browser_width
  • browser_input_capture
  • webgl_enabled
  • clickable_exists
  • clickable_add
  • clickable_add_ext
  • clickable_change
  • clickable_change_ext
  • clickable_set_style
  • clickable_delete
  • analytics_event
  • analytics_event_ext
  • os_browser
  • http_get_request_crossorigin
  • http_set_request_crossorigin

除了 HTML5 目标模块的这些函数外，还有一个特殊的 JavaScript 变量，可以在 html5game 文件夹的内容存储在完全不同的位置（URL）的极少数情况下使用。通常，此路径在 JavaScript 文件中是硬编码的，并且与调用 JavaScript 的位置相关，但你可以在调用游戏脚本之前设置变量 g_GameMakerHTML5Dir，游戏将从新定义的位置运行。比如：

<script>var g_GameMakerHTML5Dir = "http://mygamealternatedirectory.com/";</script>
