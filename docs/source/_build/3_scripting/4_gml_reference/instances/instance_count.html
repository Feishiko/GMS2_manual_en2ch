instance_count


描述

使用此 只读 变量，你可以获得房间中所有活动实例的数量。这将包括运行代码的实例，但 不 包括那些已使用 instance_deactivate 函数冻结的实例。请注意，此函数仅提供步的 开始（start） 处的实例数，因此不会考虑在步开始后对房间中的实例所做的任何更改。


语法：

instance_count;


返回：

Real（实数）


例如：

if (instance_count < 100)
   {
   var dif = 100 - instance_count;
   while (--dif > 0)
      {
      instance_create_layer(random(room_width), random(room_height), "Effects", obj_Star);
      }
   }

上面的代码将创建物体 “obj_Star” 的多个实例，直到总实例数达到100。
