instance_position_list


描述

使用此函数，可以检查位置是否与其他物体所有实例发生碰撞。使用此选项时，你将在房间中为物体的所有实例检查一个点，如果该物体选中了精确碰撞，则将对实例的边界框或实例的遮罩进行检查。函数将用被认为与调用实例碰撞的物体的所有实例的唯一 ID 值填充（以前创建的）数据结构列表（DS list），你可以根据从被检查的位置到被发现碰撞的实例的原点的距离来排序列表。请注意，该函数还接受特殊关键字 all，在这种情况下，将列出找到的要处于碰撞状态的所有实例。函数返回找到的实例数，如果找不到，则返回 0。


语法：

instance_position_list(x, y, obj, list, ordered);

参数 描述
x 用于检查实例的 x 位置。
y 用于检查实例的 y 位置。
obj 要检查的实例的物体。
list 用于存储碰撞实例的 ID 的数据结构列表。
ordered 列表是（true）否（false）应按距离排序。


返回：

Int（整数，发现碰撞的实例数）


例如：

var _list = ds_list_create();
var _num = instance_position_list(mouse_x, mouse_y, all, _list, false);
if _num > 0
    {
    for (var i = 0; i < _num; ++i;)
        {
        instance_destroy(_list[| i]);
        }
    }
ds_list_destroy(_list);

上面的代码将检查是否与鼠标位置上找到的所有实例发生碰撞。这些将被添加到一个数据结构列表中，然后循环通过该列表来销毁碰撞中的每个实例。
