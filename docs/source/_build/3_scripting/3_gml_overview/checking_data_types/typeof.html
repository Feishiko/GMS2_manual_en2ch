typeof


描述

此函数将任何给定变量的数据类型作为字符串返回。可能的返回值列在下表中：

字符串 描述
number 该变量为一个实数（浮点）数
string 变量为一个字符串
array 该变量引用一个数组
bool 该变量为一个布尔值（true / false）
int32 该变量为 32 位整数
int64 该变量为 64 位整数
ptr 该变量为一个指针
undefined 该变量未定义
null 变量为一个空值（正常情况下不应该看到）
vec3 该变量为 3 值向量
vec4 该变量为 4 值向量
unknown 值未知。永远不 应该看到它，如果看到它则表示在最基本的层面上出现了错误，例如内存覆盖


语法：

typeof(variable);


参数：

参数 描述
variable 获取变量的数据类型。


Returns:

字符串（见上表）


例如：

data_str = typeof(global.ExtensionInput);

上面的代码获取给定全局变量持有的数据类型，并将该字符串返回给实例变量。
