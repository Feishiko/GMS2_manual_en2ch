is_ptr


描述

此函数返回给定变量是否为指针。在某些情况下，你需要检查并查看变量在 GameMaker Studio 2 中保存的数据类型，此时应使用此函数。它返回 true 或 false，具体取决于值是否为指针。有关数据类型的更多信息，请参见 此处。


语法：

is_ptr(n);


参数：

参数 描述
n 要检查的参数。


返回：

Boolean（布尔值）


举例：

if !is_ptr(val)
   {
   show_debug_message("Not a valid texture!");
   }

上面的代码检查变量 “val” 以查看它是否包含指针，如果不是，则它在调试控制台中显示一条消息。
