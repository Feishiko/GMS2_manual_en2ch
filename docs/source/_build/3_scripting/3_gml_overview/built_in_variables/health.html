health（弃用）


语法：

health;


返回： Real（实数）


描述

此变量在 全局 范围内，通常是用于保存玩家健康状况的数值。请记住，并不一定总是这样 - 只是因为它被称为 “健康值” 并不意味着它必须用于健康...它可以保存你希望用于游戏的任何实数型或整型值。

重要！ 此变量仅用于支持以前版本的 GameMaker 中的旧项目，不应在新项目中使用，因为它已弃用，并且在将来的版本或更新中可能会完全删除。


举例：

if place_meeting(x, y, obj_Enemy)
   {
   health -= 10;
   if health <= 0 instance_destroy();
   }

上面的代码检查当前实例以查看是否与 “obj_Enemy” 索引的物体实例发生冲突，如果是，则从全局生命值变量中删除 10 点，然后检查该值以查看它是否小于或等于 0。如果是实例会摧毁自己。
