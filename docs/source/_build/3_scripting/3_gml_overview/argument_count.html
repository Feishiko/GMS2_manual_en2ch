argument_count


描述

该只读变量返回传递给脚本的参数个数。一般和参数数组（argument[0...15]）同用以允许对给定脚本输入数量可变的参数。


语法：

argument_count;


返回：

实数


举例：

var i, arg;
for (i = 0; i < 5; i += 1;)
   {
   if argument_count > i
      {
      arg[i] = argument[i]
      }
   else
      {
      arg[i] = -1;
      }
   }

上述代码使用argument_count变量来初始化变量，并示范了如何允许脚本接收可变数量的参数。
