设置主音量


描述

通过此动作，你可以设置整个声音系统的音量。The volume value you give can be between 0 (silent) and 1 (full volume) and the scale is linear, such that a value of 0.5 would be half volume. 请注意，此操作将影响当前在房间中播放的所有声音的所有实例，并将影响所有播放声音的所有未来实例。此动作也与任何单个声音的音量设置无关，因此如果您将声音设置为以0.5的音量播放然后将主音量设置为0.5，则声音将以0.25音量播放。
