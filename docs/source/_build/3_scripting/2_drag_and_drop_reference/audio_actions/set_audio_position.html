设置音频的播放位置


描述

通过此动作，你可以设置一个声音开始播放的特定时间点。给定要播放的声音，以及开始播放的位置，但这个动作不会为你播放声音，它只是设置声音的播放位置。要使用此动作，你必须首先设定位置然后使用播放音频动作播放声音，否则在下次播放相同的声音资源（将从新位置开始）之前，你的播放位置不会改变。位置设置以秒为单位 - 你可以使用小数值 - 你可以使用获取音频长度动作获得总声音长度，并且如果你提供的值大于声音的长度，这将会覆盖前面的位置，所以如果声音是10秒长，你给12秒作为开始位置，声音将从2秒开始。
