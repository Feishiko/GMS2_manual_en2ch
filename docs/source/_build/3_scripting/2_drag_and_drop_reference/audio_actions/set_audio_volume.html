设置音频音量


描述

通过此动作，你可以设置给定声音的音量。从资源浏览器中选择声音，然后设置音量值。音量可以在0（静音）和1（全音量）之间，并且音阶是线性的，这使得0.5的值是半音量。请注意，此动作将影响当前在房间中播放的这个声音的所有实例，并将影响所播放声音的所有未来的实例。

音量 声音新的音量（从0到1，默认为1）
