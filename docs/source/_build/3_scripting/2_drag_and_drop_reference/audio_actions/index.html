音频动作库


GameMaker Studio 2有一个完整的音频引擎，基于* .ogg，* .mp3和* .wav声音格式。添加到IDE的这些类别的声音可以通过下面的函数使用。对于比基本的音效更复杂的事物或者播放单个音乐片段，你可以参考高级的音频函数，你可以用它们修改声音的播放方式。此外，还有一系列专用函数，专门用于流式传输的音频和定位音频（ 3D声音效果以及分组音频）。

IMPORTANT! When using audio on the HTML5 target, you should be aware that not all browsers support Web Audio and so may not play any sound for your project when run. You can get a general idea of Web Audio support from the following link: Can I Use Web Audio?.


这个音频函数库有有以下动作：

播放音频
停止播放音频
停止所有音频
暂停音频
暂停所有音频
恢复音频的播放
恢复所有音频的播放
设置音频的播放位置
获取音频长度
设置音频音调
获取音频音调（Get Audio Pitch）
设置音频音量
获取音频音量
设置主音量
获取主音量
如果音频正在播放
如果音频被暂停
