描述

上述代码块设置了 图像混合 颜色，来和实例精灵“混合”。默认值为-1（表示白色），这将使用不使用混合颜色绘制精灵。任何其他值，包括内置颜色常量例如 c_red或 c_aqua将 混合 精灵和给定的颜色。请注意，要使此更改可见，实例必须 <i0>没有</i0> 绘制事件(这样 <i1>GameMaker Studio 2</i1> 将默认绘制精灵) 或使用 <a2>自我绘制</a2> 动作。It is important to note too that you should try to limit blending on the HTML5 platform (unless using WebGL), as each blended sprite has to be cached separately and so having many blended sprites may adversely affect performance.
