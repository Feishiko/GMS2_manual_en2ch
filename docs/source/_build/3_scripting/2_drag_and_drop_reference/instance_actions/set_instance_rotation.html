设置实例旋转


描述

上述代码块将为实例设置 图像角度（旋转） 。旋转角的单位是度，默认是0，向右是0度，向下是90度，向左是180度，向下是270度。将此变量设置为0以重置实例, 这意味着它将以在精灵编辑器中定义的形状被绘制。. 请注意，要使此更改可见，实例必须 <i0>没有</i0> 绘制事件(这样 GameMaker Studio 2 将默认绘制精灵) 或使用 <a2>自我绘制</a2> 动作。
