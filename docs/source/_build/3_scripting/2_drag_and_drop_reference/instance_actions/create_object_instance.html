创建实例


描述

通过此动作，你可以从资源树创建任何对象的实例。你需要给出对象的名称，和要创建对象的房间内的 xy 位置。你可以勾选 "相对" 标志，这样设置的坐标位置将是相对于执行此动作块的实例的。无论什么时候你在游戏中创建了一个实例，它将被赋予一个独特的ID，这样你就能方便地在以后动作中调用此实例。因此你可以定义”目标“变量来存储此ID值。你可以在下面的例子中看到这个演示。注意这个目标变量可以是一个临时（本地）变量，这样这个变量将在此动作中被创建（译者注：这样你就不用在此动作前特意定义一个变量），这个变量可以用于之后的动作。

y 创建实例的房间中的 Y位置 (可以相对于调用实例)。
