设置实例缩放


描述

上述动作块将为 实例设置图像X轴比例 和 图像Y轴比例 。此输入值将修改实例上精灵的缩放比，默认是1，即没有缩放的原比例（1：1），更小的值意味着图像将被缩小（例如0.5意味着只有原来一半），大一点的值意味着图像将扩大。你也可以将其设为负数来翻转精灵。将其设为-1将只翻转精灵而不缩放。你也可以将水平比例和垂直比例设为与当前值相关联，例如加上或减去新值。

注意此值的改变也将影响检查碰撞的方式，除非你提供独立的 mask_index （碰撞遮罩）。这需要在 物体编辑器种设置。请注意，要使此更改可见，实例必须 <i0>没有</i0> 绘制事件(这样 <i1>GameMaker Studio 2</i1> 将默认绘制精灵) 或使用 <a2>自我绘制</a2> 动作。
