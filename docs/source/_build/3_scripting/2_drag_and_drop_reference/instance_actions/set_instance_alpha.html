设置实例透明度


描述

上述代码块将为实例设置 图像alpha 。图像 alpha 是用于绘制分配给实例的任何精灵的 "透明度" 值, 其值介于0和1之间, 其中0将是完全透明的, 1 将是完全不透明的。请注意, 如果 sprite 具有透明或半透明像素，则其透明度将按此值缩放。 因此， 如果 sprite 中有大约50% 的透明像素，并将实例 alpha 设置为 0.5, 则透明像素的最终 alpha 将为25%。. 你可以设置此值为和原来的值相关联。

请注意，要使此更改可见，实例必须 没有 绘制事件(这样 GameMaker Studio 2 将默认绘制精灵) 或使用 自我绘制 动作
