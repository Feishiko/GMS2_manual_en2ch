设置计时器的倒计时


描述

上述动作块将使一个计时器开始倒计时，当这个计时器到0时，将触发关联的Alarm Event。你需要提供计时器的编号，从0到11，然后为计时器设置一个值。此值就是在你想触发相应事件前经过的"游戏步数”。你也可以将这个值设为于之前的值相联系（相加或相减）。请注意, 该值必须是一个整数值, 您可以将其设置为-1 以停止计时。注意当计时器的数值到达0时就会触发相应事件。但计时器仍然会继续变为-1，作为它完成计时的标志。将一个计时器设为0将 不会 运行相应事件，尽管它会在下一步变为-1。如果你想某事立即发生，应该将计时器设为1。注意你可以点击 <i0> 加号图标来多次设置额外的计时器。
