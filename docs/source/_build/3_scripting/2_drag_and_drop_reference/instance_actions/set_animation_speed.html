设置动画速度


描述

此动作代码块为此实例上的精灵设置 图像速度 (也就是动画速度) 。尽管你可以在 Sprite Editor 设置这个值，但你也可以用此动作修改它，默认的动画速度是1，这意味着游戏默认动画是一帧每秒。但你可以将此值设为大于1的值，这样动画将会播放得快一些。（也许有些帧会跳过）。你也可以将此值设为小数，这意味着动画会播放得更慢。你也可以将此值设为负数，这样动画就会倒着放。
