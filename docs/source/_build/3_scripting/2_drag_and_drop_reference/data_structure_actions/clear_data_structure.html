清空数据结构


描述

此动作将清除给定数据结构中的所有值。首先必须选择要清除的数据结构类型，然后给出数据结构的索引（索引是在最初创建数据结构时返回并存储在变量中的值）。一个被清除的数据结构被认为为“空”的（即：它没有条目）， 除了网格数据。对于网格，此操作将简单地将所有网格单元格清除为0。

注意：这不会从内存中释放数据结构，只会清除信息。要释放分配给数据结构的内存，必须使用该动作释放数据结构。
