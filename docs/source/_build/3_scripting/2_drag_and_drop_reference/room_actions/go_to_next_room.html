前往下一个房间


描述

此动作将会使你的游戏前往游戏编译时的资源树列表上的下一个房间. 如果将要前往的下一个房间不存在，那么将会抛出错误并强制请注意此动作被调用时不会立即生效，而要等到目前的事件或脚本结束后才能生效。所以所有在此动作之后的动作会继续执行，直到目前的事件或脚本结束。
