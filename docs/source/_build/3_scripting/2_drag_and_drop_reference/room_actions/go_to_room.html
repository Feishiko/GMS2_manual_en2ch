前往房间


描述

此动作将结束当前的房间，并根据输入的索引来决定进入哪个房间。前往的房间必须存在，否则就会出错。如果前往的房间就是当前房间，那么此动作的效果和 Restart Room是一样的。请注意此动作被调用时不会立即生效，而要等到目前的事件或脚本结束后才能生效。所以所有在此动作之后的动作会继续执行，直到目前的事件或脚本结束。
