字符串转数字


描述

此动作将一个字符串转化为一个数字。你需要提供一个字符串，然后提供用来储存返回实数的目标变量 (可以标记为临时局部变量)。如果字符串中有字母或小数点，那么将不会 被转换。如果字符串以实数开头 而之后是字母，那么将会把开头的所有数字转换为实数，而忽略之后的字母。. 注意如果你需要转换的数字中有小数点， 那么他将保留小数点后两 位。你可以在以下看到这是怎样运作的：

字符串 返回值
"123" 123
"123 Hello World" 123
"Hello World" 0（默认情况下，如果字符串不能被转换，将返回0）。
"Hello World 123" 0
"123.45678" 123.46
