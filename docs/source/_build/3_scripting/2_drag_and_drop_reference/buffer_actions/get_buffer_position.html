获取缓冲区位置


描述

此动作可用于检索读取/写入缓冲区的当前位置。创建完缓冲区时，读/写位置设置在0字节的地方，当你向缓冲区添加数据时，此位置将按顺序移动，这意味着你随时可以使用此动作来获取缓冲区设置到的字节位置。你提供创建缓冲区动作返回的唯一缓冲区ID值，然后给出一个目标变量来返回位置值。目标变量可以标记为临时局部变量，这样它将为你创建，并在当前动作脚本或事件结束时再次从内存中删除。
