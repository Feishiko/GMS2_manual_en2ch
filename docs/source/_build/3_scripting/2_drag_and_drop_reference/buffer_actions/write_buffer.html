写入缓冲区


描述

此动作可用于将数据写入缓冲区。提供要写入的唯一缓冲区ID（由 创建缓冲区动作返回）然后写入值，可以是实数或字符串，但请注意所有值都将存储为字符串，所以当你从缓冲区读取时（使用读取缓冲区）你可能需要使用数据类型动作中的一个将它从字符串转换回原始值。请记住，读取和写入缓冲区是按照顺序进行的，因此，写入缓冲区会将“搜索（seek）”位置移动到一个位置，这意味着要写入各种数据，你只需将此动作调用所需的次数。
