搜寻缓冲区


描述

此动作可用于移动（搜索）缓冲区内的位置。你提供创建缓冲区动作返回的唯一缓冲区ID值，然后是要移动到的以字节为单位的偏移值。偏移值将应用在缓冲区的开始位置，所以调用此函数会将读/写位置移动到缓冲区的开头，然后移动给定的字节数并在那里定位新的读/写位置。但是，如果将偏移标记为相对，则它将从当前位置向前移动读取/写入位置（译者注：偏标标记的相对是指relative，最新版本里“寻找”标签的下拉列表有三项，即开始，相对，结束；向前移动是指move forward，向前进；如果偏移为负值，则向后移动，也就是向后退的意思）。
