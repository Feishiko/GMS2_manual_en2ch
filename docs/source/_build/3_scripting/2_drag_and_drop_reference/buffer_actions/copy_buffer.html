复制缓冲区


描述

此动作可用于将存储在一个缓冲区中的所有数据复制到另一个缓冲区。当使用两个缓冲区并从一个缓冲区复制到另一个缓冲区时，必须使用创建缓冲区动作提前创建好它们，并提供返回的唯一缓冲区ID作为from/to的参数。

注意：无法复制到同一缓冲区。
