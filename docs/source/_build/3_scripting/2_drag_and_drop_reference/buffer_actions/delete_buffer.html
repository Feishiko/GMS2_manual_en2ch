删除缓冲区


描述

此动作可用于从内存中删除缓冲区，释放用于创建它的资源并删除它当前可能包含的任何数据。你提供创建缓冲区动作返回的唯一缓冲区ID值。请注意，你可以通过单击动作旁边的加号图标选择多个要删除的缓冲区，在其中添加另一个缓冲区的ID。

注意：当你不再需要那些动态创建的资源，始终要记得从内存中删除它们，以防止内存泄露。
