如果存在缓冲区


描述

此动作可用于检查给定缓冲区ID是否有效以及缓冲区是否实际存在于内存中。如果缓冲区ID有效且存在，则动作将返回true（真）否则它返回false（假）。你可以标记“not”以使该动作检查是否有缓冲区不存在。
