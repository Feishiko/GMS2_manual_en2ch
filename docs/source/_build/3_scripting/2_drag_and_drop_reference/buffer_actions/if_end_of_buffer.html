如果是缓冲区的结束点


描述

此动作可用于检查读/写位置是否已到达缓冲区的末尾。提供创建缓冲区动作返回的唯一缓冲区ID值，然后如果读/写位置已到达缓冲区中所有可用数据的末尾，动作将返回true（真），如果还没有则返回false（假）。你可以标记“not”以使该动作检查读/写位置不在缓冲区的末尾的情况。
