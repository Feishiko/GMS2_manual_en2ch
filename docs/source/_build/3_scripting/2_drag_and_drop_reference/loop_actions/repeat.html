重复循环（Repeat）


描述

此动作将重复任何动作，重复次数是给定的。你需要给出动作重复的次数。然后你需要给出到底是那些动作需要重复，就像你将动作添加到“if”中那样。例如：把动作块放在动作的一侧而不是下面。

被添加到此repeat动作一侧的动作将被包含在此repeat动作中。当repeat动作全部执行完后，才会继续执行repeat之后的动作。（请看下面的例子）。如果你想在循环中任何时刻结束循环，你只需要调用 Break 动作。
