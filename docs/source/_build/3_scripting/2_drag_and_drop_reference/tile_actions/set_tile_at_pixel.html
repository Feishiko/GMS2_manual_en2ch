设置图块（像素）


描述

使用此动作，你可以在给定层上给一个图块设置新索引。在此动作中你首先需要提供目标图块所在层的名字（字符串），这在房间编辑器中是已经定义好了，然后提供房间中的x,y轴坐标来取得图块的图块数据。此X和Y位置可以被设置为和调用动作的实例位置相对。被设置的图块是指在房间指定位置中的图块，而用来设置的图块是指在房间编辑器中被分配给层的图块。最后你需要提供图块的索引来设置图块（图块索引将按从左上到右下的顺序排序，0将是最左上）。注意你可以将图块索引设为0来清除此图块。
