获取图块数据变换


描述

使用此动作，你可以确认某图块的图块数据是否经过翻转，镜像或旋转。. 此图块数据是一个值，它反应了此图块的索引，翻转，镜像和旋转值，以及任何自定义的图块遮罩。你在使用此动作之前需要首先使用获取单元格内的图块数据l 或 在指定像素获得图块数据如果图块数据确实经过变换，那么将返回 true 否则返回 false 。返回值将被存储在目标变量中，此变量需要之前就被创建了，或你选中临时(temp)按钮来创建的新临时变量。如果你需要更多关于图块数据的信息，参阅 图块地图-图块函数.
