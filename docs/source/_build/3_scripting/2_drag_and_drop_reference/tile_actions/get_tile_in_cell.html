获取图块（网格单元）


描述

使用此动作你可以在一个指定层上的一个图块索引。在此动作中你首先需要提供目标图块所在层的名字（字符串），这在房间编辑器中是已经定义好了。然后提供层上此图块单元格的行以及列。最后你要提供将要存储返回的索引的目标变量。如果你设置的目标变量是 临时（局部）变量，那么此动作将创建变量来保存返回值，直到事件结束。

参数：

参数 描述
层 需要取得图块所在的层
列 此单元格所在列数（从左到右）
行 此单元格所在行数（从上到下）
目标 用来存储返回值的目标变量
