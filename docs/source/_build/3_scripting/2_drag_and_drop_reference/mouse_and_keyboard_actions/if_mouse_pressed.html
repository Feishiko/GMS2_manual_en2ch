如果鼠标按住


描述

使用此动作你可以检测当前鼠标按钮是否被按下。当鼠标按钮被按下（并且按住）时，此动作会在每一步（step，或者译为帧更贴切？）返回 真，除此之外的情况则返回 假 。此外，如果你勾选否选项，则可以检测鼠标按钮是否 没有被按住。在这种情况下，此动作会在鼠标按钮没有被按下时返回 真，在被按下时返回 假 。如果你只需要检测鼠标单击（而不是按住），请使用动作 如果鼠标点击.

如果 if 的判断结果是 真。那么此动作将会马上执行。放在别处的动作将会在此 if 动作块后执行。
