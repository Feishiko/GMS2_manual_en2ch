如果键位点击


描述

此动作检测键盘键被按下的最初状态（即键盘点击）在键位按钮点下的瞬间，它将返回 真 ，其它时间返回假。如果你勾选了not 选项，此功能将会相反，也就是说，动作会检测键盘键是否 没有被点击，如果被点击了就返回假，其余情况返回真。 .
