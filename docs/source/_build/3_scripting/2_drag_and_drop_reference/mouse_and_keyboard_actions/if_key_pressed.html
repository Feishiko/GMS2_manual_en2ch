如果键位按住


描述

使用此动作你可以检测当前键盘按钮是否被按住。当键盘被按住时，此动作在每一步会返回真，除此之外的情况返回假，此外，如果你勾选否选项，则可以检测键盘按钮是否 没有被按住。在这种情况下，此动作会在键盘按钮没有被按下时返回 真，在被按下时返回 假 。如果你只需要检测按键单击（而不是按住），请使用动作 如果键位点击.
