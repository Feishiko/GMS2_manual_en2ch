设置文本对齐


描述

使用这个动作，可以设置之后所有绘制文本动作的字体对齐方式。你可以设置水平对齐方式为居左，居中或居右。垂直对齐方式为居上，居中或居下。文字将根据绘制动作的坐标去对齐，如下图所示：

文字水平排布，左对齐。
文字水平排布，居中对齐。
文字水平排布，右对齐。
文字竖直排布，上对齐。
文字竖直排布，中对齐。
文字竖直排布，底对齐。

注意：设置将对之后的所有绘制字体动作生效。所以，举例来说，你可以再游戏开始的任何事件中调用该动作（并不限制只能用绘制事件）调用后，之后所有的文字都会使用你设置的对齐方式。 不过，如果你需要在游戏中调用多种对齐方式，你需要在每次绘制不同的对齐方式之前调用这个动作。
