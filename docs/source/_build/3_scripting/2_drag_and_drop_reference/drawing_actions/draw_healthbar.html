绘制血条


描述

使用这个动作，你能绘制一个颜色条，用来图示值的大小。要显示的数值应该是一个百分比值，数值在0-100之间。但你可以用公式 (CURRENT_Value / MAXIMUM_value) * 100来取一个较大的值来显示。例如，你可以用它来显示能量，健康值，法力，时间等等任何适合用进度条来显示的东西。除了绘制的值之外，你还可以设置绘制血条的方向（从左到右或从右到左，从上到下或从下到上）以及位置，绘制位置可以在房间中的指定位置，或者是调用这个动作的实例所在的位置。除了位置之外，你还可以设置血条4个角的混合用颜色，创建出从红到绿的渐变色（只是举例，你可以使用任何颜色）。下图展示了设置不同参数的血条的显示效果：

参数 描述

右 血条右端的位置

背景 血条背景颜色
轮廓 血条外轮廓的颜色
