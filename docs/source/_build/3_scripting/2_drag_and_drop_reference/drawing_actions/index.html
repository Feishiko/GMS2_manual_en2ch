绘制动作库


在绘制动作库中你可以找到绘制精灵，文字，形状的动作，在这些动作中还可以设置绘制属性。 大部分动作 只能被物体使用于各种 绘制事件中，如果放在绘制事件之外将不能生效。有一类动作是例外，是设置动作，它们可以在任何事件中调用，对之后绘制的所有实例生效。

请注意：如果你添加了任何动作到物体的主要 绘制事件中，这个物体产生的实例将 不再绘制它被赋予的精灵，除非你额外告诉 GameMaker Studio 2 去绘制精灵,使用绘制自身这个动作。一般来说，只要绘制事件中的动作为空，GameMaker Studio 2 会自动绘制赋予给实例的精灵。

绘制自身
绘制值
绘制变幻的值
绘制精灵
绘制变形精灵
绘制堆叠的精灵
绘制矩形
绘制渐变矩形
绘制矩形
绘制渐变椭圆
绘制线条
绘制血条
绘制实例分数
绘制实例的血条
绘制实例生命数
设置绘制颜色
获取绘制颜色

获取绘制透明度
