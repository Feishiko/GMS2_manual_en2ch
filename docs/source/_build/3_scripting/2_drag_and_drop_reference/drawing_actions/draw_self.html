绘制自身


描述

当你向对象的绘制事件添加任何类型的操作或代码时，你告诉 GameMaker Studio 2你在接管 ， 你会告诉它从现在开始，之后该绘制什么。（默认情况下，如果在主要绘制事件中没有动作，实例会绘制分配给它的精灵）这意味着当你添加了绘制事件后，如果你不额外调用绘制动作，精灵将不再被绘制。而绘制自身动作则是简单的绘制这个物体被赋予的精灵，绘制时可以在先前事件中设置的任何变换（如缩放，混合或透明度变化）。

注意：这个动作只能用于各种 绘制动作，如果用在其它位置将不生效。
