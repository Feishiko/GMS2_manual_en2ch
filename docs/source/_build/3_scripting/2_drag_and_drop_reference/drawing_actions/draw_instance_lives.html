绘制实例生命数


描述

此动作会绘制一个精灵去展示实例当前的每条生命。绘制数量等于实例的变量 生命数的值，当你的实例使用了任何需要生命数的动作时，这个特殊的变量“生命数”会自动创建在这个实例内。 你需要给出绘制精灵的顺序，可以是行（水平，从左到右），或者列（垂直，从上到下），同时也需要给出绘制的位置。位置可以是房间内的绝对位置，或者是调用这个动作的实例所在的位置。绘制间距默认为精灵的宽度或高度。请注意，绘制的是一个静态图。如果精灵是个序列帧图，将只绘制精灵的初始图像（第0帧），精灵后面的图片将被忽略，同时，实例通过改变变量带来的任何变形(例如 图片的x轴变换 或者 图片混合)也将被忽略。标题文本会在生命数之前绘制，如果它是一串文本，则应该用“”引起来。标题文本和生命数都会使用当前的字体，颜色和段落格式去绘制。

注意：这个动作只能用于各种 绘制动作，如果用在其它位置将不生效。

参数 描述

x 绘制位置的x坐标
y 绘制位置的y坐标
