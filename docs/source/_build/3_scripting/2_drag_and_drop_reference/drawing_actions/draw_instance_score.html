绘制实例分数


描述

你可以使用这个动作绘制实例的分数。当你在GameMaker Studio 2使用了 任何 分值动作，调用分值动作的实例将生成一个新的实例审阅变量称为 分数 这个变量可以像其他变量一样，可以被检索或设置。这个动作会在屏幕指定位置绘制这个变量以及其它你想添加的标题文本。标题文本会在分数值之前绘制，如果它是一串文本，则应该用“”引起来。标题文本和分数都会使用当前的字体，颜色和段落格式去绘制。

注意：这个动作只能用于各种 绘制动作，如果用在其它位置将不生效。
