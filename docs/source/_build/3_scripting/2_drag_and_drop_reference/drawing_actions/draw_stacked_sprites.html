绘制堆叠的精灵


描述

此动作将在房间内的给定位置一个接一个地绘制一个精灵给定数量的精灵你需要指定需要绘制的精灵，以及堆叠指令，指令可以是 行 （水平，从左到右）, 或者列 (垂直，从上到下),，以及需要绘制精灵的次数，和绘制的位置。位置可以是房间内的绝对位置，或者是调用这个动作的实例所在的位置。绘制间距默认为精灵的宽度或高度。注意，此动作只能绘制静态图像，如果是动态精灵的话，只会绘制第0帧，第0帧之后的帧数会被忽略，任何对精灵的变形也会被忽略（例如 x轴变形或者颜色混合).

注意：这个动作只能用于各种 绘制动作，如果用在其它位置将不生效。

参数 描述
精灵 待绘制的精灵

数字 需要绘制精灵的个数The number of sprites to draw
x 绘制的x坐标
y 绘制的y坐标
