绘制圆角矩形


描述

此动作可以在房间中的指定位置绘制椭圆，使用一组混合颜色来形成渐变。你需要指定椭圆的左上角和右下角位置，绘制动作将把椭圆绘制在它们之间。位置可以是房间内的绝对位置，或者是调用这个动作的实例所在的位置。你可以设置颜色从椭圆的边缘或中心混合。你可以点击底部 填充框来设置是否要填充椭圆。

注意：这个动作只能用于各种 绘制动作，如果用在其它位置将不生效。

参数 描述

右 右侧开始绘制的位置
底 底部开始绘制的位置
中心 中心颜色
