设置绘制颜色


描述

使用这个动作，你可以设置之后将绘制图形的颜色（绘制图形，例如矩形、椭圆、线）或者绘制文字的颜色。注意：不需要每次绘制动作前都使用这个动作，可以只在游戏开始设置一次。举例来说，假如你的所有文字都希望是黄色的，你可以第一个房间的创建事件中调用这个动作，之后所有的文字（和图形）都会用黄色绘制。此外，如果你需要多次使用不同的颜色，每次更换颜色前需要调用这条动作。

此动作在底端有一个勾选框 使用透明度. 这样，设置颜色动作就也可以同时设置绘制的透明度了。效果与动作 设置绘制透明度 一致。如果你不希望绘制时有透明度，指向用固定颜色绘制，可以取消勾选该选项。（默认是勾选的）。
