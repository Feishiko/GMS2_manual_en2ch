绘制值


描述

使用这个动作，你可以绘制一串文字说明和/或一个变量或者一个值到屏幕上。caption参数是一个字符串，你可以把它加载值的前面，也可以留空。不过，如果你希望包含文字的话，需要用引号引起来。值可以是变量，固定值或另一个字符串-不过再这种情况下你需要把它放到引号中，并且不可为空。然后你要给出字符串绘制的位置，位置可以是房间内的绝对位置，或者是调用绘制动作的实例所在的位置。注意：文字绘制效果与当前指定的字体，颜色，对齐方式一致。以上说的所有值都可以在绘制之前重新指定，如下面的例子所示：
