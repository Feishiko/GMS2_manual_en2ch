绘制精灵


描述

此动作在房间内绘制精灵。你需要指定精灵的位置（位置可以是房间内的绝对位置，或者是调用这个动作的实例所在的位置。），需要绘制的精灵的帧，以及是否需要绘制多帧。（注意，帧数值应该是0以及0以上）如果是一个动态精灵，你希望像把它赋予物体一样绘制，使用-1作为帧索引。或者建造一个实例变量 image_index, 注意，如果实例没有被赋予精灵，将不会动。如果被赋予的精灵帧数和需要绘制的精灵不一致，绘制时会绘制额外的帧或者丢失一些帧。

重要此动作无法将精灵变形绘制，即使你修改了图片的混合属性或比例. 如果需要在绘制时修改精灵，请使用动作绘制变形精灵.
注意：这个动作只能用于各种 绘制动作，如果用在其它位置将不生效。
