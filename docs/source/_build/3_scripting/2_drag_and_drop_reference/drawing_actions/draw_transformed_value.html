绘制变幻的值


描述

使用这个动作，你可以绘制一串文字说明和/或一个变量或者一个值到屏幕上。caption参数是一个字符串，你可以把它加载值的前面，也可以留空。值可以是变量，固定值或另一个字符串-不过再这种情况下你需要把它放到引号中，如果需要，可以留空。然后你要给出字符串绘制的位置，位置可以是房间内的绝对位置，或者是调用绘制动作的实例所在的位置。此外，你可以设置文字比例，沿水平轴或垂直轴拉伸它，以及旋转（旋转方向是逆时针的，所以如果你设置90°旋转，将向左旋转90°。设置-90°则是向右旋转）缩放由给定参数决定，所以，如果给宽度的缩放值是0.5，将把文本宽度 减半 ，相似的，如果宽度缩放值设置为2，将会把宽度加倍。默认缩放值为1。注意：文字绘制效果与当前指定的字体，颜色，对齐方式一致。以上说的所有值都可以在绘制之前重新指定，如下面的例子所示：
