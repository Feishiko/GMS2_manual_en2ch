绘制渐变矩形


描述

此动作在房间内指定位置绘制一个渐变矩形，绘制时可以设置渐变颜色。你需要指定矩形左上角的坐标，以及右下角的坐标，然后矩形将在它们之间被绘制。坐标可以是房间内的绝对坐标，或者是调用此动作的实例所在的位置。然后，你需要设置矩形每个角的渐变颜色，以及这个矩形是被填充还是只勾边。设置方式是勾选底部的填充单选框。

注意：这个动作只能用于各种 绘制动作，如果用在其它位置将不生效。

参数：

参数 描述
左 左侧开始绘制的位置
顶 顶部开始绘制的位置
右 右侧开始绘制的位置
底 底部开始绘制的位置
左上 左上角的渐变颜色
右上 右上角的渐变颜色
左下 左下角的渐变颜色
右下 右下角的渐变颜色
