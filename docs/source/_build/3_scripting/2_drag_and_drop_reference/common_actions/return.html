return


描述

在创建 动作块脚本 时，此动作用来返回一个值。返回值可以是任何一个变量存储的东西，比如字符串，数值，指针或资源ID，甚至是变量本身。注意当我们调用返回动作时，此脚本将会退出，此脚本中返回动作后的部分也不会被执行。就算你在此 事件 还有其它的动作， 如果你在调用脚本后，仍然有动作，那么这些动作将会被执行。
