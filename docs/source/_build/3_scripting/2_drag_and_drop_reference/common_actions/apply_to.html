Apply To...


描述

此动作可以用来改变此动作作用域 使将来所有动作与此动作发生联系。像 if 动作，如果你把之后的动作都放到应用于（Apply to...） 的右边，那么他将在物体和实例的作用域来执行。当你点击此动作右上方可爱的小小箭头，那么你就能在弹出的目录中选中作用域，这样你就能使一组不同的物体，或一个单独的实例（你需要给出实例的ID）来在当前物体继续执行代码前，执行多个动作。

注意如果你要把动作添加进应用（Applies to），这些动作应该被放进动作的一边 ，具体请看下图：

这些动作将在被“应用于”选中的物体或实例上执行，其它的动作将在“应用于”中的动作后执行。
