宏常量


描述

此动作允许你定义 宏常量。不要被名字吓着，他和普通常量差不多。此宏常量就只是一个独特的 常量 值，或者可以计算出独特常量值的表达式。此宏常量和普通常量的使用方法是一样的。. 你也可以使用一个可以生成一个返回值的函数，这样此常用常量将每次被调用时将会执行此函数。注意此宏常量的作用域是 全局 ，可被用于任何实例，脚本或代码块。但在被定义后宏常量 不能被更改 。注意你可以点击 <i0> 加号图标来多次定义不同的宏常量。更多信息请点击 这里.
