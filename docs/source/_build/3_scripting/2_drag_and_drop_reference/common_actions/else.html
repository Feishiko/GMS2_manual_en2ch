描述

此"else" 动作 只是用于与 If变量 一起使用。当if返回false时，将会执行else语句。例如你可以使用“if"语句来检查玩家的血量是否小于等于0。如果是的话将会销毁此实例。但如果你想在玩家学量 不 小于等于0时执行一些动作，那么你就在if后加上else语句，只有当if返回 false时才执行else语句。
