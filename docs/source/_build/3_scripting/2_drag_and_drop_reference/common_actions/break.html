break


描述

此动作用来跳出 循环 或 switch（switch是函数的一种）。在循环中，只有当执行到最后一个语句时才会结束，但你可以调用 break 来使它提前结束。比如本来某事要重复10次，但你用此动作可以使它执行不满10次就结束。在switch中，语句将终止于最后一个 case，否则执行完当前case后将继续执行下一个case，但你可以调用 break 使其执行完特定的case后就结束。如果你学过编程中的循环语句将会更好的理解这一点。

请注意它不 会退出脚本或结束当前事件（如果你要这样做请使用 exit语句）。 它只是简单跳过当前循环或switch代码块，并继续执行之后的代码或事件。注意在循环语句外使用 break 将会导致错误。
