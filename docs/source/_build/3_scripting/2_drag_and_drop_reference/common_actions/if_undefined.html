描述

此代码用来检测一个变量或表达式是否返回了一个未定义的值。如果此变量或表达式返回的值未定义，那么此动作返回 true 否则返回 false。. 有多种情况能使返回值为未定义，例如没有初始化变量。这些未定义的值可能会造成严重的错误甚至游戏崩溃。所以检查它们是很有必要的。如果你选中了“NOT”参数，那么结果将会反向。例如一个变量返回值<i0>不是</i0>未定义时，将会返回true。
