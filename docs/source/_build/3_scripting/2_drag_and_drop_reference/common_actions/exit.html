exit


描述

此动作将根据之前函数的执行情况来决定是否执行。（可用在if后，功能类似break）。如果它在一个事件中被调用，那么它将会直接退出此动作而执行此事件后的动作代码块。但如果它在一个 动作代码脚本中被调用，那么它将只退出此脚本，而返回到使用此脚本的事件中，继续执行此脚本后面的语句。如果你的脚步原来有返回值，但你在此脚本中调用了此动作，那么它将 不会 有任何 return 。
