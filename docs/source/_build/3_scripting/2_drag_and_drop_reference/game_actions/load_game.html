载入游戏


描述

此动作将加载一个之前已经保存的游戏，后者使用了Save Game 动作。你需要提供需要加载文件的名称（字符串格式，包括扩展），然后游戏将会被加载并从保存点继续运行。此动作并不是一个全游戏的保存/加载系统，如果你尝试保存一个正在运行的游戏并加载他，他可能会报错，尤其是你使用了 数据结构 或 粒子。所以，你不能随时随地保存或加载游戏，你只能从一个单独的游戏物体中加载和保存游戏（例如检查点）。请注意此动作被调用时不会立即生效，而要等到目前的事件或脚本结束后才能生效。所以所有在此动作之后的动作会继续执行，直到目前的事件或脚本结束。为了更容易理解加载存储游戏数据的方法，请看 文件动作。
