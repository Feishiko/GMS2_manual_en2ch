重启游戏


描述

使用此动作你可以重新开始游戏。这与第一次开始游戏相同，所以 其它-游戏开始事件 将会被触发， 其它-游戏终止事件也会被正常触发。但是你应当注意到，全局变量不会自动初始化（除非你用代码让它们执行了初始化）——这意味着 它们仍将保持之前的值，数据结构 和 粒子 将不会被释放，所以你需要 在调用重新开始游戏动作前 或 在游戏终止事件后，手动设置初始它们。请注意此动作被调用 不会 立即生效，而要等到目前的事件或脚本结束后才能生效。所以所有在此动作之后的动作会继续执行，直到目前的事件或脚本结束。
