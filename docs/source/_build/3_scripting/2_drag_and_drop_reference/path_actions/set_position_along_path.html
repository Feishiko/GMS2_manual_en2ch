设置路径沿线位置


描述

此动作将会设置一个实例在路径的位置。该值在0到1之间，因此如果此值为0.5，则实例将被移到路径的正中间。注意你必须让该实例为特定路径调用 开始跟随路径 动作，否则此动作将不会产生任何影响。
