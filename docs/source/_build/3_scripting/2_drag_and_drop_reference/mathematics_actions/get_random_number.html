获取一个随机数


描述

此动作将在给定的最大和最小值之间产生一个随机数。你可以选择产生数的种类是小数还是整数，然后设置产生数字的最大值和最小值。这个随机值将会被返回给目标变量，此目标变量可以是一个临时局部函数。.

注意： When generating a random integer the action will return an integer value between the minimum and maximum value inclusive, so a range of 5 - 10 could return 5 or 10 as well as any integer in between. 但如果产生小数，它将不会返回最大值。例如产生5到10之间的小数，它将产生5或小于10但不包括10的数字。注意它仍可以返回9.99999999这样的数。
