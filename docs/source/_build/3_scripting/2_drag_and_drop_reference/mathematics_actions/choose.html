使用此动作你可以提供不同类型的参数，可以是一个真的数字，一个字符串，一个资源ID（resource ID)或一个常量，动作将选择其中一个返回。你可以点击动作旁边“增加”图标，为其添加额外的选项，并给出其它可以返回的值。返回值将被存储在目标变量中，此变量需要之前就被创建了，或你选中临时(temp)按钮来创建的新临时变量。
