项目图像生成器


可以从 GameMaker Studio 2 IDE 顶部的 工具菜单 访问 项目图片生成器。此工具旨在从一组图像中快速创建所有不同目标平台所需的不同的初始屏幕、促销图像和图标。你提供基本图像和基本图标，然后单击 生成 按钮，每个平台的 游戏选项 将使用新图像进行更新。

为了使其正常工作，你需要提供高分辨率启动画面和高分辨率图标图像。建议的尺寸为：

  • 启动画面：纵向至少为 1920 x 1080 像素，或 1080 x 1920 像素，并且为 24 位无透明度 PNG 格式

  • 图标：至少 1024 x 1024px（这是所需的最大图标的大小），并且为 24 位 可以 有透明像素的 PNG 格式

提供这些图像后，它们将显示在预览窗口中，然后你可以设置在将它们应用到不同平台时要使用的 “填充” 类型。你可以选择以下内容：

  • 保持长宽比例 - 这将缩放图像以适应保持长宽比并添加背景颜色以填充侧面（如果填充不是 1:1）

  • 拉伸 - 这将拉伸图像以适合图像 / 图标的完整大小。

  • 裁剪以适应 - 这将得到基本图像，使它最适合沿水平轴或垂直轴，然后裁剪其余部分。例如，如果你的 640 x 480 图像必须用作 480 x 800 的闪屏，则图像将按比例 放大 以适合垂直轴（480 -> 800），图像的其余部分将被裁剪掉。同样，如果你的 1080 x 1920 图像必须适合 800 x 800，那么图像将按比例 缩小 以适合水平轴，其余图像将被裁剪掉。

现在，你只需单击 Generate，GameMaker Studio 2 将为每个平台填充所有游戏选项 图像 和 图标 设置的图像。

重要！ 虽然此工具将生成你需要的所有图像，但这并不意味着它是完美的！ 它将尽力使图像适合每个平台的不同尺寸和分辨率，但有时这可能看起来不像你真正想要的，因为它确实没有 “一刀切”。 因此，在运行此工具后，我们建议你修改每个平台游戏选项中的所有图像，并手动调整那些可能看起来不像你想要的图像。
