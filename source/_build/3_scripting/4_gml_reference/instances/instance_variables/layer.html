layer


Description

This built-in variable is created for every instance in a room and contains the layer ID value of the layer that the instance is assigned to. This value can then be used in other functions like layer_get_depth or it can be changed to move the instance to another layer, but note that if the layer being assigned does not exist in the current room, then you will get an error that will force your game to close.

You can assign this variable either the unique ID of the layer to move to or the name string of a layer from the room editor. For example, if you have a layer in the room editor called "Foreground", you could have either of the following sets of code:

layer = "Foreground";

// or...

layer = layer_get_id("Foreground");


You can use the layer variable in any function that requires a layer ID.


Syntax:

layer;


Returns:

Layer ID value (Real)


Example:

layer = layer_create(-1000);

The above code will create a new layer with a depth of -1000 and then set the instance layer variable to the returned layer ID, moving the instance from the layer it is currently on to the new layer being created.
