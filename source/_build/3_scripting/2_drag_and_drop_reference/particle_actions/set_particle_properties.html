Set Particle Properties


Description

This action will permit you to set the speed, direction and friction properties for a given particle type. You first specify the particle type ID value (which should be the same as the value used when you created the particle using the Create Particle Type action) and then you can set the different ranges for both speed and direction. As with other properties (like particle lifespan), you give a minimum and maximum value for speed and direction and particles will be emitted with a random value somewhere in between.

Speed is calculated as pixels-per-game-frame, so a minimum setting of 2 and a maximum setting of 5 will emit particles moving at anywhere between 2 and 5 pixels-per-game-frame. Direction works the same way, where you give a minimum and maximum value and the actual particle direction will be a random value between the two. Note that for both speed and direction you can use negative values for the minimum values, and that direction in GameMaker Studio 2 is calculated with 0° to the right, 90° to the top, 180° to the left and 270° to the bottom, so if you want a random direction covering the right side of the emitter you would set direction minimum to -45 and maximum to +45.

Finally you can set a friction value for particles. This is the amount that a particle will reduce it's speed each game frame, and as such is usually set to a smaller value like 0.1. This value is subtracted from the overall particle speed as it updates.


Action Syntax:


Arguments:

Argument Description
Type ID The ID value for the global type to set the properties of
Min Speed The minimum speed for a particle (in pixels per game frame)
Max Life The maximum speed for a particle (in pixels per game frame)
Min Life The minimum direction for a particle (in degrees)
Max Life The maximum direction for a particle (in degrees)
Friction The friction to use for the particle (in pixels per step)
