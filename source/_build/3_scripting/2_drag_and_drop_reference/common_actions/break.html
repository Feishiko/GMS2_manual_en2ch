break


Description

This action is used to "break out of" loops or switches. In a loop, it is used to end the loop before the final conditions are met, for example if you have set something to repeat 10 times and then call break in the repeating code block, it will end the repeat loop before the full 10 repetitions are completed.With switches it is used to signify the end of a specific case, although it can be omitted in which instance the next case will be run and so on until either the switch is ended or a break is encountered.

It's important to note that this does not exit the script or end the current event (for that use exit), but instead simply skips to the end of the current loop or switch code block and then continues running any code that comes afterwards until the end of the script or event. Also note that using a break outside of these contexts will cause an error.
