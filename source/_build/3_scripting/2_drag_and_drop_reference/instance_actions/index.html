Instance Actions Library


The Instance action library is where you can create and destroy instances as well as change their properties and use their alarms. The available actions are all listed below:

设置实例旋转
设置动画速度
设置实例缩放
设置实例透明度
设置实例颜色
如果实例存在
调用父对象事件
