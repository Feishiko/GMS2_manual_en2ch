脚本

这一章节讨论GameMaker Studio 2向你提供的用于编程的不同的脚本语言。创建你的项目使用的语言或者方法取决于你的技能和你以前的技术背景，既然每个人都是不同的， GameMaker Studio 2致力于满足你的不同需求，为你打造游戏提供了两种不同的选项：

拖拽式操作（DnD™）适合初学者或者那些视觉或艺术定位的开发者（这有时也可以称为“视觉脚本语方言”）。由拖拽动作组成拖放到物体编辑器中，创建一系列的物体实例将遵循的函数列表。在手册的以下章节，你可以找到 DnD™ 的使用方法：
  1. 拖拽按钮概述
  2. 拖拽式操作参考

在这里你可以看到GameMaker语言语法的概述，以及使用方法和程序结构的例子，你还可以找到所有内置GML函数的完整的参考指南。如果你是脚本使用的新手，建议从GML概述开始，不然只需要使用参考指南去查找所需要的函数名就可以了：
  1. GML 概述
  2. GML 参考

值得注意的是，制作游戏时它们相互之间并不排斥，你可以混合使用GML和DnD™，根据你的技能和要求，随心所愿。
