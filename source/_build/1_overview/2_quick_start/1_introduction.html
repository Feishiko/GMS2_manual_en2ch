介绍


GameMaker Studio 2 是一个游戏开发工具， 旨在让您和您的团队根据游戏设计的思路以最快，最直观的方式制作游戏，并且能跨平台开发。它主要是用来制作2D游戏的工具，虽然制作3D游戏也是可行的。有许多工具和编辑器能帮助实现你的游戏想法，将您的最终项目从同样的初始基础资源移植到多个平台上。

对于那些编程世界的新手或者从没有使用过游戏创建工具的人， GameMaker Studio 2 提供了一个直观且易用的 Drag and Drop (DnD™) 动作界面图标，允使你使用虚拟的代码逻辑创建你自己的游戏。如果你是一个更有经验的程序员，你更适合使用脚本语言 GML 创建你的游戏，或者你甚至可以结合两者，使用DnD™动作调用函数，在你写代码的时候，给你极大的灵活性。使用直观时尚的布置，在IDE中将你游戏的不同方面"链接"在一起，给你一个独特并且直观、可视化的工作流。

你也将发现这个IDE拥有不可思议的灵话性，它可以根据你的需要进行定制 - 默认的IDE设置不覆盖你要求的东西 - 通过 插件系统的使用。而且，默认 GameMaker Studio 2 已经自带一些强大的工具，比如一个完整的 图片编辑器 可以用来创建你自己的精灵或图块集，也可以导入外置编辑器完成的图片， 包括Spine 骨骼动画 和 SWF 格式 矢量文件。有了图片之后，在 物体编辑器绑定到物体上就是简单的问题了，添加一些代码或 DnD™ 动作，然后将物体的实例放进 房间编辑器的游戏房间里。还有一些额外的工具，用于创建 路径、 时间轴、 着色器以及其它更多的东西。

这个帮助文件将告诉你关于 GameMaker Studio 2所有你需要知道的东西，以及你怎样在创建你的游戏时使用它们。. 然而，请意识到，就算有像 GameMaker Studio 2这样的游戏制作工具，设计电脑游戏也不是毫不费力的 ，所以没有 "给我制作一个MMORPG" 这种按钮！你也要意识到，要制作一款优秀的游戏还包含许多方面：游戏性，美术，音乐，用户交互等.... 但是，最重要的方面，就是耐心。而 GameMaker Studio 2是一个强大的工具，在能随心所欲的制作游戏之前，我们先从一些简单的小例子开始。你就会明白，制作游戏是一件很有趣的事情， 而且学习使用 GameMaker Studio 2 也是非常值得的！
