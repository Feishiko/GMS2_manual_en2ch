起始页


第一次打开 GameMaker Studio 2，你将被要求登录你的 YoYo 帐户（如果你没有就注册一个 - 看 这里）。一旦注册，登录，GameMaker Studio 2 IDE 主界面将打开起始页：

在顶部，你能看到最基本的IDE菜单，以下手册将对各节进行解释：

  1. 文件菜单
  2. 编辑菜单
  3. 构建菜单
  4.   窗口菜单
  5. 工具菜单
  6. 市场菜单
  7. 帮助菜单
  8. 布局菜单

请注意根据你目前焦点所在的窗口，顶部菜单栏将出现 特定环境的 菜单选项。例如，如果焦点在资源树窗口，额外菜单项将出现“Resources（资源）”。下面的章节将说明起始页的其它内容：

当你导航至起始页的不同页面，点击将带你回到主页。


起始页的左右你可以看见能打开的以前的项目。鼠标放在上面，你可以看到完整的项目名称和路径，点击 将打开这个项目。


起始的顶栏包含许多图标链接，使导航更容易。列出如下：

  • 我的帐户 - 登录到你的YoYo帐户（查看 这里获取更多详细信息）

  • YoYo 游戏 -将你带到YoYo 游戏 主网站。

  • 社会媒体图标 - 这些将你带到各种YoYo游戏t的社会媒体网站，那里有 GameMaker Studio 2到目前为止最新的消息。

你可以在项目部分创建、打开或导入项目。你可以点击 新建 创建一个新的项目，打开 打开一个外部项目或者 导入t 打开一个 GameMaker: Studio 1.4 GMX 项目文件，或者打开压缩的 YYZ GameMaker Studio 2 项目文件。打开或者导放都将打开一个文件浏览器，你可能浏览到你需要的文件。

注：GameMaker Studio 2 不完全向后兼容GameMaker: Studio 1.4 项目，但是导入1.4项目 应该 仍然可以运行，因为废弃函数已经自动使用 兼容脚本重建了。可能出现的问题和GML所做的更新的全部细节，请参阅帮助中心文章 Porting A GMS 1.4 Project To GameMaker Studio 2（导入一个GMS1.4项目到GMS2），以及手册的废弃函数部分。

要创建一个新的项目，仅需要点击按钮标签New (新建)，你将看到下面的屏幕内容：

GameMaker Studio 2 允许您指定希望创建的项目类型，你可以根据你的编码知识或习惯来选择。可用的选项有：

  • New Drag and Drop Project: 这将创建一个新的项目，其中 对象编辑器 (以及允许代码的其他编辑器) 将被设置为使用 GameMaker Studio 2 Drag and Drop (拖拽) 界面。更多关于 Drag and Drop(DnD™)的详细信息请查看 这里. 请注意，这不是唯一的DND™接口，你可以切换 DnD™ 和 GML 代码，使用鼠标右键.

  • New GameMaker Language Project: 这将创建一个项目，默认程序使用的是GameMaker Language (GML). 关于 GML 更多的详细信息在这里. 请注意，这不是唯一的GML接口，你可以切换 DnD™ 和 GML 代码，使用鼠标右键 .

无论你选择什么， 然后提示选择保存项目的位置，并在 GameMaker Studio 2中打。


在起始页的信息部分允许您浏览 YoYo 游戏市场 以及浏览和打开的YoYo Games团队关于GameMaker Studio 2 具体操作方面的教程和演示（更多信息见下文）。



教程和演示


当你在开始页面点击 Tutorials 或Demos 按钮时，跳转到显示教程和演示页面。显示你自己的以及可供下载的官方教程演示:

下载任何官方教程或演示资料，只需点击其中一个图标它会自动下载到你的 "Owned（拥有）"中，(注意：在此之前你首先需要登录到YoYo市场). 一旦教程出现在 "Owned（拥有）" 部分，当鼠标在某个教程之上时，会显示它的信息说明。

一旦教程或演示已经安装，你可以点击打开它，选择保存位置，之后在IDE中打开这个教程。注意：安装教程和演示将有额外的图标 可用于从您的计算机卸载该教程(它不会从你的 "Owned（拥有）" 列表里删除)。

在起始页的教程演示部分也有几个按钮，帮助您更好的使用和浏览您的资产:

单击将从市场刷新项目列表，以便您可以检查，看看是否有任何已更新，并获得最新版本。
单击此按钮切换你自己的教程和那些已安装的视图。

需要注意的是通过一个教程工作时，你可以在任何时候保存项目并关闭IDE甚至改变项目，而不必担心失去你的当前位置。

还值得一提的是，你可以从Marketplace找到第三方的教程和演示，一旦这些已购买并添加到您的帐户，它们也将出现相应的页面的“Owned(拥有)”部分。
