GameMaker Studio 2 许可证


GameMaker Studio 2有许多不同的许可证类型，选择哪一个在于导出平台的受众是什么。下面是每一种许可证的详细说明。

GameMaker Studio 2 最开始是以试用版许可信息启动的，而后你可以依据你的需求购买升级包。试用版具有一些限制但是允许你体验我们最重要的特性和样例工程。最值得注意的是，使用试用版的许可信息你可以在你的机器上测试游戏，但是不能在任意一个目标平台上创建最终的可执行程序包。. 此外，在可用的资源上，你也会受限。

  • 物体 - 15
  • 精灵 - 20 (no dynamic loading permitted, no SWF or Spine sprites permitted)
  • 声音 - 10 (no audio buffers permitted)
  • 图块集 - 2
  • 脚本 - 10
  • 路径 - 5 （不允许创建动态路径）
  • 时间轴 - 2 （不允许创建动态时间轴）
  • 字体 - 5 （不允许创建动态字体）
  • 房间 - 5（不允许创建动态房间）
  • 着色器 - 0
  • 包含文件 - 0
  • 扩展 - 0
  • 配置 - 0


除了这些资源限制以外，还有一些元素会在IDE中消失：

  • 你不可以创建纹理页
  • 你不可以创建纹理组
  • 你不可以创建音频组
  • 你不可以通过右键菜单添加已有资源
  • 你不可以使用内置的源代码控制
  • 你不可以从市场下载资源
  • 你不可以导入旧版GameMaker Studio的工程
  • 你不可以从YYZ工程包进行导入

使用桌面版的GameMaker Studio 2将会移除所有试用版许可的限制（见上文），并且你可以在下列目标的桌面平台上编译工程：

  • Windows
  • Mac OS
  • Linux (Ubuntu)

All targets come with two compile options, one is the VM that uses a runner and interpreted code and the other which is the YYC (YoYo Compiler) which creates native code and a compiled executable. Note that the YYC option will require extra setup. For more information see the section on Required SDKs or the helpdesk section on Desktop Targets.
