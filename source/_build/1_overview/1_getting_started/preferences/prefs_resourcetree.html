Preferences - Resource Tree


The Resource Tree Preferences are used to define certain properties for how the resource tree and it's elements function. The main options are:

  • Maximum number of resources open simultaneously before warning: You can select multiple resources for opening at a time by using + to select them and then right clicking and choosing open all. If you have selected more than the number specified in this option, then you will be shown a warning, since having too many resources open at one time can cause issues with performance on lower end machines. The default value is 20.

  • Open the image editor when double clicking on a sprite resource: By default double clicking on any resource will open that resource for editing, but often when dealing with sprites, after the initial setup when you create the new resource, you open it to edit the image rather than the sprite properties. Selecting this option will open the image editor for the sprite resource instead of the sprite resource itself. You can still open the sprite editor by right clicking and selecting Open Properties. This option is off by default.

Here you can select the behaviour of the confirmation dialogues when working with a resource from the resource tree. The available options are:

  • Default for resource deletion: When you delete a resource from the resource tree you normally get a warning asking if you are sure you want to proceed. By setting this option to "Okay" the message will be suppressed as if you had clicked the "Okay" button to proceed. The default value is "Show Message".

  • Default for opening multiple resources: As mentioned above in the general Resource Tree preferences, when you open multiple resources at once from the resource tree, you can get a warning if the number exceeds the number specified. By setting this option to "Okay" the warning will be suppressed as if you had clicked the "Okay" button to proceed. The default value is "Show Message".
