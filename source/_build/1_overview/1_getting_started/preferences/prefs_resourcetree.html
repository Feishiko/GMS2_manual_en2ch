偏好设置 - 资源树


资源树偏好设置用于为资源树和它的元件定义某些属性。主要选项有：

  • 发生警告前同时打开资源的最大数目（Maximum number of resources open simultaneously before warning）：你可以使用 + 同时选择多个资源然后右键点击并选择打开全部（open all）一次性打开。如果你选中的数目超过选项中的数目，你将会看到警告信息，因为同时打开太多的资源在低配电脑上会出出一些性能上的问题。默认值为20.

  • 双击精灵资源打开图像编辑器（Open the image editor when double clicking on a sprite resource）：默认双击任何资源都将打开编辑，但是经常在处理精灵的时候，你创建新的资源的初始化设置之后，你打开它编辑图像而不是精灵属性。选中这个选项将打开这个精灵资源的图像编辑器，而不是精灵资源本身。你仍然可以通过右键点击并选择打开属性打开精灵编辑器。该选项默认为关。

在处理资源树的资源的情况，这里可以选择确认对话框的行为。可用的选项有：

  • 删除资源的默认设置（Default for resource deletion）：当你删除资源树的一个资源，通常你会得到一个警告，询问你是否想要继续。通过设置这个选项为“确定（Okay）”，消息将会被阻止，就好像你已经点击了“确定”继续了。默认值为“显示消息”。

  • 打开多个资源的默认设置（Default for opening multiple resources）：就如上面提到的，在资源树的基本偏好设置中，同时打开的超过指定的数目你会获得一个警告。通过设置这个选项为“确定（Okay）”，警告消息将会被阻止，如果已经点击了“确定”。默认值为“显示消息”。
