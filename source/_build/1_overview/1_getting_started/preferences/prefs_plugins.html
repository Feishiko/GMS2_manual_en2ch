偏好设置 - 插件


插件偏好设置用于控制已经添加到GameMaker Studio 2 IDE的任何额外的插件。默认有一个专门的子项源码控制插件整合GameMaker Studio 2项目与Git工具。

本节讨论要求让 Git SCM在GameMaker Studio 2上能够使用的偏好设置。首先合并工具部分包含以下字段：

  • 启用详细日志记录（Verbose Logging）：Git 启用详细日志记录输出日志。该选项默认为关

  • 合并工具路径（Path to Merge Tool）：在此处指定 Git 合并工具的完整文件路径。

  • 合并工具选项（Options for Merge Tool）：在此字段你可以添加任何额外的命令，在使用合并工具时都将运行。

设置完合并工具后，你可以使用这些选项设置用户身份（User Identity）：

  • 用于提交的用户名（Username to commit as）：在这里你必须输入你将做任何提交到Gi时所使用的用户名（它必须同Git仓库所指定的用户名一致）。

  • 用户密码（Password for username）：在这里，你必须输入上面提供的用户名关联的密码。

  • 用于提交的邮箱（Email to commit as）：与此Git身份关联的邮箱。
