偏好设置 - 样条曲线（Spline）编辑器


样条曲线编辑器偏好设置主要用来定义路径编辑器的外观和感受，尽管也将会影响IDE的特定的其它方面（比如物理形状编辑器）。存在以下选项：

  • Insert path point precision (px) - This sets the precision value (in pixels) for inserting points on a path, ie: the mouse must be within this range to insert the point rather than add a new one. 默认20px。

  • Anchor point width - Set the width of the anchor points, in pixels. 默认8px。

  • Anchor point height - Set the height of the anchor points, in pixels. 默认8px。

  • Path colour - The colour to be used to show the path as you edit it. 默认值是（rgba）#FFFFFFFF。

  • Highlighted path segment colour - The colour to be used to highlight any selected path segments. 默认值是（rgba）#FFFFFFFF。

  • Invalid path colour - The colour to be used to highlight a path that is not valid. 默认值为（RGBA颜色） #FF0000FF。

  • Anchor points colour - The colour to be used to draw the path anchor points. 默认值是（rgba）#C0C0C0FF。

  • Invalid anchor points colour - The colour to be used to draw the path anchor points. 默认值是（rgba）#800000FF。

  • Selected anchor point colour - The colour to be used to draw the path anchor points that have been highlighted (selected). 默认值是（rgba）#00A4FFFF。

  • Selected invalid anchor point colour - The colour to be used to draw the anchor points that are highlighted but invalid. 默认值是（rgba）#FFA400FF。

  • Mouse-over anchor point colour - The colour to be used to draw the anchor points that are active. 默认值是（rgba）#FFFFFFFF。

  • Mouse-over invalid anchor point colour - The colour to be used to draw the anchor points that are active but invalid. 默认值为（RGBA颜色） #FF0000FF。

  • Mask overlay colour - Set the overlay colour for masks if they are enabled (see above). 默认值为（RGBA颜色） #404040FF。
