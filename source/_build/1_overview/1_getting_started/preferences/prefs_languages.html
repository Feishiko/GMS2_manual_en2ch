偏好设置 - 语言


语言偏好设置（Language Preferences）被用于定义编程时的一些语言特性将会被如何显示在GameMaker Studio 2 的IDE中。GML的选项如下所示

  • 在状态栏中显示参数类型（Display argument types in status bar）：该选项关系到在你使用你自己的被JSDoc标注过的自定义脚本（Script）时是否显示各种参数等……（详情请见这里）当该选项被勾选并且你在脚本中标注了参数类型的时候，它们会被显示在状态栏当中，并被花括号括起来。该选项默认为关

  • 在状态栏中显示参数描述（Display argument descriptions in status bar）：勾选该选项之后你在自定义脚本中使用JSDoc标记给出的描述都会在编辑器窗口底部的状态栏中显示出来。该选项默认为关

  • 显示系统函数的美式拼写（Show US spelling for system functions）：默认地，GameMaker Studio 2会显示函数的英式拼写，例如draw_text_colour，然而这可能不是你所习惯的拼写风格，因此你可以使用这个选项切换到美式拼写，这样一来，就会显示成draw_text_color了。该选项默认为关
