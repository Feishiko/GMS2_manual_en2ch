偏好设置 - 房间编辑器


房间编辑器偏好设置用于定义房间编辑器的外观和感受。基本设置如下：

  • 链条透明度开始淡出的像素距离（The distance in pixels that the chain alpha begins to fade out）：设置链条开始淡出的与父窗口的距离。默认值为10像素。

  • 链条透明度停止淡出的像素距离（The distance in pixels that the chain alpha stops fading out）：设置链条连接结束淡出的与父窗口的距离。默认100像素。

  • 衰减停止距离的链条透明度（The chain alpha at the falloff stop distance）：设置链条透明度淡出衰减点的最终透明度的值。默认是0.3。

  • 层之间深度的间距（The amount of spacing for depth between layers）：房间编辑器创建多个层时，将会基于层列表的位置自动绑定一个深度。深度的增加量根据上面给定的的值来确定。默认值是100，所以每一层的深度将是前一层的深度加100。

  • 新房间的默认宽度（Default width of a new room）：设置初始的宽度作为新房间默认宽度。默认是1024像素。

  • 新房间的默认高度（Default height of a new room）：设置初始高度也将作为新创建房间的默认高度。默认是768像素。

  • 创建新房间时添加默认的层（Add default layers when creating a new room） - 当你创建任意一个房间资源，GameMaker Studio 2将为你生成两个“基础”的层， - 一个背景层和一个实例层。你可以切换这个行为，让新房间不会产生预制的层。该选项默认为“开”。

  • 打开的房间显示视野按钮的初始化设置（The initial setting for the Show Views button on opening a room）：如果一个房间里有激活的视野，这个可以通过房间里视野边界框可以看出来，这个区域是否可见可以通过专用的按钮进行切换。但是你也可以选择在打开房间的时候通过取消选中此选项让视野边界不可见（仍可用此按钮再次显示它们）。默认值是打开，使打开房间时视野边界可见。

  • 添加到新房间的默认背景层颜色（The colour for the default background layer added to new rooms）：设置初始的颜色，选择作为默认新的背景层的颜色。默认是（rgba）#00000000。

  • 继承图块的高亮颜色（Highlight Colour For Inherited Tiles）：对于已经继承自一个父房间的图块，设置它的高亮颜色。默认是（rgba）#FF000050。

  • 房间内锁定的项目的选择框颜色（Selection frame colour for locked items in the room）：设置环绕被锁定的资源的选择框的颜色。默认是（rgba）##AAFFFFFF。

  • 房间内未锁定项目的选择框的颜色（The properties frame colour for unlocked items in the room）：设置房间编辑器中环绕未锁定的资源的选择框的颜色。默认颜色是（rgba）##AAFF3333。

  • 房间里锁定项目的属性框的颜色（The properties frame colour for locked items in the room）： 设置环绕被锁定的资源的属性框的颜色。默认是（rgba）##AAFFFFFF。

确认对话框偏好设置用于更改房间编辑器执行某些操作时消息框设置。你可以更改的消息包括：

  • Default for automatically updating layer depths - When you change layer depths a message is shown to warn you that doing this will also automatically update all the other layer depths. With this option you can choose to have the message show, have it update the layers as if you'd clicked "Ok", or have it not update anything as if you'd clicked "No".

  • Dismiss layer locked warning - If you try to delete a layer in the room and it happens to be locked, you will be shown a message by default. However setting this option to "Ok" will not show the message and just not let you delete the layer.

  • Dismiss item locked warning - If you try to delete an item it happens to be locked, you will be shown a message by default. However setting this option to "Ok" will not show the message and just not let you delete the item.

  • Dismiss item's owner layer locked warning - If you try to delete an item on a layer and the layer happens to be locked (even if the item is not), then you will be shown a message by default to inform you that it can't be done. However setting this option to "Ok" will not show the message and just not let you delete the item.

  • Dismiss invalid resource being dragged into room error - If you try to add a resource to a layer that is not designated as being for that resource type (like adding a sprite to a path layer), then you will be shown a message by default to inform you that it can't be done. However setting this option to "Ok" will not show the message and just not let you add the resource.

  • Default for parent room item deletion - Here you can choose what happens when you delete an item that is part of a parent room (and so present in all child rooms). The default is to show you a warning message and let you decide what to do, but you can set it to "Delete All" to suppress the message and go ahead with the item delete from the parent room and all its children, or you can select "Keep All" to suppress the message abort the delete.

  • Default for parent room layer deletion - Here you can choose what happens when you delete a layer that is part of a parent room (and so present in all child rooms). The default is to show you a warning message and let you decide what to do, but you can set it to "Delete All" to suppress the message and go ahead with the layer delete from the parent room and all its children, or you can select "Keep All" to suppress the message abort the delete.

  • Dismiss dragging resources with multiple or no selected layers - Here you can choose what happens when you drag a resource onto the room editor when you have selected multiple layers in the layer editor (or have not selected any layer at all). The default is to show a warning message but you can set this option to "OK" to suppress the message and continue, without adding any resources.

  • Dismiss dragging resources into a locked or invisible layer error - Here you can choose what happens when you drag a resource onto a layer that has been locked or is flagged as invisible. The default is to show a warning message but you can set this option to "OK" to suppress the message and continue, without adding any resources.

The Grid Settings preferences are used to change how the grid overlay works, with the following options available:

  • Horizontal pixel spacing - This will set the horizontal space for gird cells of the room grid in pixels. Default value is 32px.

  • Vertical pixel spacing - This will set the vertical space for gird cells of the room grid in pixels. Default value is 32px.

  • Display Grid - This option will toggle on or off the room grid when you first open a room. It is on by default, but even if it is switched off you can enable it again using the appropriate button in the room editor itself.

  • Snap to grid - With this option you can set whether a room will have assets snap to the room grid by default or not when a room is initially opened. It is on by default, but even if it is switched off you can enable it again using the appropriate button in the room editor itself.

  • Line Colour - This option permits you to set the colour (including the alpha component) of the room grid lines. The default value is (rgba) #FFFFFF30.
