gesture_get_double_tap_time

This function is used to get the time it takes between two touches/clicks to trigger a Double Tap Gesture event. The time is measured in seconds and has a default value of 0.16.

 

Syntax:

gesture_get_double_tap_time();

 

Returns:

Real (seconds)

 

Example:

if gesture_get_double_tap_time() != 0.16
    {
    gesture_get_double_tap_time(0.16);
    }

The above code checks to see if double tap time for gestures is set to 0.16 seconds and if it is not it sets it to that value.
