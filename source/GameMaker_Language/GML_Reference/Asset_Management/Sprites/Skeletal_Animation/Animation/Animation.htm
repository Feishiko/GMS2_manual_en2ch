Animation

When working with skeletal animation sprites created using Spine, you can have multiple animations per-sprite, and as such you need to be able to switch between them at appropriate times in your game. To help you do that, GameMaker Studio 2 provides the following functions:

 

  • skeleton_animation_get
  • skeleton_animation_set
  • skeleton_animation_get_ext
  • skeleton_animation_set_ext
  • skeleton_animation_get_duration
  • skeleton_animation_mix
  • skeleton_animation_list
  • skeleton_animation_clear
  • skeleton_animation_get_frames
  • skeleton_animation_get_frame
  • skeleton_animation_set_frame
