array_pop

With this function you can retrieve a value from the end of an array without having to know the length of the array. You supply the variable that holds the array, and the function will return the contents of last entry in the array. If the array is empty, the function will return undefined.

 

Syntax:

array_pop(variable);

Argument Description
variable The variable that holds the array.

 

Returns:

Value (any data type) or undefined

 

Example:

var _lastscore = array_pop(score_array);
draw_text(32, 32, "Last Score = " + string(_lastscore));

The above code will pop the last value from the given array then draw it along with some text to the screen.
